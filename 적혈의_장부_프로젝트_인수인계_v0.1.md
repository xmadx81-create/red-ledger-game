# 적혈의 장부 프로젝트 인수인계 문서 v0.1

작성일: 2026-06-14
프로젝트명: 적혈의 장부
부제: 뱀파이어 운영 시뮬레이션

---

## 1. 한 줄 소개

《적혈의 장부》는 공익 혈액기관을 가장한 뱀파이어 가문의 비밀 혈액 수급망을 운영하며, 신뢰와 은폐, 생존 사이에서 선택을 내리는 다크 행정 누아르 운영 시뮬레이션 게임이다.

홍보 문구:
낮에는 생명을 나누고, 밤에는 생존을 계산한다.

---

## 2. 핵심 콘셉트

장르:
- 경영 시뮬레이션
- 선택형 스토리
- 조직관리
- 다크 행정 누아르

세계관:
- 실제 기관명은 사용하지 않는다.
- 표면 조직: 백십자재단 산하 혈연센터
- 이면 조직: 카르테인 가문 혈액관리국
- 플레이어는 카르테인 가문의 신임 운영책임자다.
- 낮에는 공익기관처럼 운영하고, 밤에는 뱀파이어 가문의 생존을 위해 혈액 수급망을 관리한다.

핵심 질문:
- 질서를 유지하기 위해 어디까지 숨길 수 있는가?
- 생존을 위해 얼마나 인간성을 포기할 수 있는가?
- 공익과 착취의 경계는 어디인가?

---

## 3. 1차 MVP 범위

목표:
- 7일 동안 혈액 수급망을 운영하는 짧은 플레이 가능 프로토타입

플레이 시간:
- 15~25분

주요 화면:
1. 타이틀 화면
2. 운영 대시보드
3. 인물 관리
4. 사건 선택
5. 엔딩 화면

주요 콘텐츠:
- 사건 카드 12개
- 주요 인물 5명
- 엔딩 5개
- 로컬 저장 방식 우선
- 추후 Supabase / Google Sheets / Notion / OpenAI API 연동 가능

## 6. 주요 인물

### 서윤재 실장
- 역할: 총괄 관리자 / 카르테인 가문 혈액관리국 실장
- 성향: 냉정, 질서중심, 감정 배제
- 기능: 조직 불안 안정화, 위기 대응
- 위험: 잔혹한 해결책 선호

### 김도현 과장
- 역할: 야간 근무 의사
- 성향: 피곤한 현실주의자
- 기능: 혈액 품질 관리, 오염 혈액 발견
- 위험: 윤리적 갈등 누적 시 내부 고발 가능

### 박세연 팀장
- 역할: 인간 물류 협력자
- 성향: 계산적, 실무형
- 기능: 운송망과 보관소 관리
- 위험: 자금 부족 시 충성도 하락

### 정하린 기자
- 역할: 탐사 기자 / 외부 위협
- 성향: 집요함, 회의적, 정의감
- 기능: 언론 노출 상승 이벤트의 핵심 인물

### 엘리엇 카르테인
- 역할: 가문 원로
- 성향: 고풍스럽고 잔혹함
- 기능: 무리한 혈액 요구, 원로회 압박 이벤트

## 7. 사건 카드 12개

1. 내부 직원이 야간 혈액 운송 기록을 발견
2. 기자가 야간 운송 차량을 촬영
3. 희귀 혈액이 예상보다 부족하다
4. 가문 원로가 특별 혈액을 요구
5. 인간 협력자가 죄책감을 호소
6. 오염 혈액이 발견된다
7. 보건청 감사 일정이 앞당겨진다
8. 신규 헌혈 캠페인이 크게 성공한다
9. 내부 직원 사이에 소문이 돈다
10. 경쟁 가문이 수급망을 침범한다
11. 시민단체가 혈액 사용처 공개를 요구한다
12. 7일차 도시보건청 감사

---

## 8. 엔딩 5개

1. 질서의 수호자
   - 가문 위신, 인간 신뢰, 혈액 재고가 안정적이고 언론 노출/조직 불안이 낮은 경우

2. 피의 독재자
   - 가문 위신은 높지만 인간 신뢰가 낮고 조직 불안이 높은 경우

3. 내부 고발
   - 조직 불안과 언론 노출이 동시에 높은 경우

4. 원로회의 숙청
   - 가문 위신과 혈액 재고가 낮은 경우

5. 새로운 공존
   - 인간 신뢰가 높고 김도현 충성도 및 특수 연구 이벤트가 성공한 경우

## 10. 현재까지 생성한 주요 에셋

기준 시안:
- 뱀파이어 헌혈 시뮬레이션 대시보드
- 헌혈의 집 밤의 헌혈소
- 헌혈의 집 인물 관리 화면
- 긴급 사건 서류 발견과 선택의 기로
- 2차 에셋 콘셉트 보드

배경:
- 비 오는 밤의 고요한 로비
- 어두운 밤의 고요한 사무실
- 어두운 아카이브 저장실 풍경

주요 인물:
- 엄격한 분위기의 남성 초상화: 서윤재 실장
- 진지한 의사의 초상화: 김도현 과장
- 세련된 비즈니스 여성 초상: 박세연 팀장
- 진지한 탐정의 초상: 정하린 기자
- 고풍스러운 귀족의 초상화: 엘리엇 카르테인

사건:
- 어두운 아카이브의 긴장감
- 비오는 밤의 비밀스러운 감시

추가 생성:
- 비 오는 밤의 고딕 건축물: 원로회/가문 회관 후보
- 혈연센터 외관: 현대 혈연센터 외관 후보
- 도시보건청 감사실: 7일차 감사 이벤트 배경

## 12. 개발 방식

1차는 외부 연동 없이 로컬 데이터/JSON으로 진행한다.

추천 구조:
red-ledger/
├─ public/assets/images/
├─ src/data/
├─ src/components/
├─ src/screens/
├─ src/game/
└─ App.jsx

추후 연동:
- Supabase: 로그인, 저장 데이터, 랭킹
- Google Sheets: 사건 카드/인물 데이터 운영 관리
- Notion: 세계관 문서와 운영자 공지
- OpenAI API: 동적 사건 생성, NPC 대사
