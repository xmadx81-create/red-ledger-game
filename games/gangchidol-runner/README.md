# 🦭 강치돌 러너 (Gangchidol Runner)

가로화면 모바일 러너 게임. 점프로 장애물을 피하며 코인을 모아 `독도사랑주유소`
글자 세트를 완성하고, 4단계(아침→낮→저녁→밤)를 완주하는 미션입니다.

- **플레이:** `index.html` 을 브라우저에서 열기 (모든 에셋이 파일에 내장된 단일 HTML)
- **조작:** `Space` / 화면 탭 = 점프, `P` = 일시정지

## 게임 규칙

| 요소 | 효과 |
|------|------|
| 🟡 일반 코인 | +5점 |
| 🔵 글자 코인 | +10점, `독도사랑주유소` 7글자 세트 완성 시 보너스(+30·+60·+90…) |
| 🚧 장애물 (라바콘·타이어·고양이·맨홀·가드레일·박스) | 충돌 시 하트 −1 |
| 💧 물웅덩이 | 감속 |
| 🍌 바나나 | 가속 |
| ❤️ 목숨 | 3개 |

점수는 브라우저 `localStorage` 에 TOP 50 순위로 저장됩니다.

## 이번 업그레이드 (그래픽/연출 강화 + 코드 리팩토링)

원본 예술 에셋(캐릭터/장애물/배경 등 16종 base64)은 **그대로 보존**하고,
그 위에 얹는 렌더링·연출을 대폭 강화했습니다.

**연출**
- 파티클 시스템 — 코인 획득 반짝임, 달리기 먼지, 착지 퍼프, 충돌 파편
- 화면 흔들림(screen shake) + 피격 비네트 펄스
- 분위기 연출 — 단계별 컬러 그레이딩 + 비네트, 떠다니는 구름·갈매기, 밤하늘 별/달, 해/노을
- 캐릭터 스쿼시 & 스트레치, 점프 높이에 따라 변하는 동적 그림자
- 부동 점수 텍스트(+5 / +10 / 보너스) + 글자코인 펄스 오라
- 단계 전환 배너 애니메이션
- 가속(스피드라인) / 감속(블루틴트·물결) 효과 시각화
- 절차적 사운드(WebAudio): 점프·코인·글자·보너스·피격·단계·클리어 + 음소거 토글
- 일시정지 화면

**리팩토링**
- 단일 IIFE를 모듈 단위로 재구성: `CONFIG / Audio / FX / Sky / Input / State /
  Spawn / Update / Render / HUD / Board / Boot`
- 게임 튜닝값(단계·점수·물리·효과)을 `CONFIG` 한 곳으로 통합 → 밸런스 조정 용이

> 기존 게임 메커니즘과 밸런스(속도/난이도/점수/판정)는 동일하게 유지했습니다.
> 추가된 콤보 표시는 시각 피드백 전용으로 점수에 영향을 주지 않습니다.
