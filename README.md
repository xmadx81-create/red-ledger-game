# 헌혈의 집 (Blood Donation House)

**밝고 명랑한 헌혈센터를 모티브로 한 수집형 카드 배틀 RPG**

> 밝은 표면, 은근한 이면. 낮에는 생명을 나누고, 카드를 모아 끝없이 더 강해진다.

## 프로젝트 개요

《헌혈의 집》은 밝은 헌혈센터를 거점으로 **혈맹 카드를 모아 레벨업·합성·강화로 끝없이 더 강한 카드를 만들어 가는, 엔딩 없는 실시간 온라인 수집형 카드 RPG**입니다.
표면은 따뜻한 지역 커뮤니티 센터, 이면은 인간과 이종족의 공생을 둘러싼 비밀과 갈등입니다.

> 이 프로젝트는 초기 운영 시뮬레이션(《적혈의 장부》)에서 **카드 RPG로 전환**되었습니다. 최신 기준은 `docs/game-design/CARD_GAME_DESIGN_001.md`(마스터 기획), 전체 현황은 `docs/game-design/MVP_BUILD_STATUS_2026-06-16.md`를 보세요.

## 핵심 루프 (전부 플레이 가능)

```text
🗺 월드맵(섬 개방) → ⚔ 스테이지 전투(5장르) 승리
  → 🎁 카드 보상·스테이지 클리어·🗂 컬렉션 기록
    → ⚗ 합성소(상위 등급 도약, 천장) / 🔒 히든 합성(비공개 최단 루트)
      → 🧪 배치 육성소(스탯 키워 자유 직업 부여 → 파티)
        → 다시 전투 …  (종료 없음)
```

상태(보유 카드·파티·진행도·컬렉션)는 `localStorage`로 화면 간 공유됩니다.

| 🗺 월드맵 (대륙>섬>거점) | `world-map.html` |
| ⚗ 합성소 (합성 가챠·천장) | `fusion-lab.html` |
| 🔒 히든 합성 (비공개 최단 루트, 데모) | `hidden-lab.html` |

## 핵심 시스템

- **유동 스탯**: 1차 8스탯(힘/민첩/지능/체력/정확도/친화/의지/행운)을 고정 총량 안에서 활동(배치)에 따라 재분배 + 잠금
- **자유 직업**: 종족 무관, 스탯 분포가 직업 후보를 산출 → 배치 육성으로 어떤 직업이든 (마법사/전사/수호자/궁수/도적/힐러/술사/정신술사 …)
- **합성(가챠)**: 카드 뽑기 없음. 등급 도약은 확률+천장. 일반 합성(공개) + 히든 트리(비공개 50% 최단 루트)
- **멀티 장르 전투**: 같은 엔진, 승리 조건만 다름. 스테이지 유형마다 다른 직업 구성이 승패를 가름
- **등급**: N · R · SR · EP · L (5단계, 세분화는 ★ 단계)

## 실행 / 검증

```bash
npm install
npm run dev        # 개발 서버 → 브라우저에서 src/web-mvp/hub.html 진입
npm run build      # 빌드 확인
npm test           # 데이터 정합성 + 프로토타입 JS 구문 회귀 스모크
npm run sim        # 경제 시뮬: 전설 1장 도달 일수 측정
```

## 톤 / 제작 원칙

- 기본 화면·카드·도감은 **밝고 명랑한 헌혈의 집** 톤. 다크 요소는 야간/특정 스테이지/스토리에만 제한적.
- 진행 원칙: **문서 → 데이터 → 밸런스 → 톤 먼저, 이미지는 마지막.** 밸런스표·체크리스트·대시보드는 이미지화하지 않는다.
- 실제 기관/적십자 계열 상표·공식 표식 사용 금지(오리지널 세계관).
