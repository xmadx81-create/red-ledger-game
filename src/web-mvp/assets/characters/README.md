# Character image assets

이 폴더는 캐릭터 도감용 이미지 썸네일을 저장하는 위치입니다.

현재 도감은 `src/web-mvp/data/character-image-map.json` 기준으로 다음 경로를 참조하도록 설계되어 있습니다.

```text
src/web-mvp/assets/characters/hero_roster_sheet.jpg
src/web-mvp/assets/characters/enemy_hero_sheet.jpg
src/web-mvp/assets/characters/staff_npc_sheet.jpg
src/web-mvp/assets/characters/support_npc_sheet.jpg
src/web-mvp/assets/characters/villain_npc_sheet.jpg
src/web-mvp/assets/characters/vampire_villain_sheet.jpg
src/web-mvp/assets/characters/golem_hero_sheet.jpg
```

바이너리 이미지 업로드가 완료되면 도감 카드의 IMAGE MAP 슬롯이 실제 이미지 썸네일로 전환됩니다.
