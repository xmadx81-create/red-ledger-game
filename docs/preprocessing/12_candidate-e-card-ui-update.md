## 등급별 시각 규칙

| 등급 | 테두리 | 아우라 |
|---|---|---|
| 일반 | 무채색 | 약함 |
| 고급 | 푸른색 | 약한 푸른 아우라 |
| 희귀 | 보라 + 청색 | 다색 아우라 |
| 전설 | 황금색 | 강한 황금 아우라 |

## 현재 인물 도감 카드 표시 정보

- 등급
- 캐릭터 ID
- 임시 초상화 슬롯
- 이름
- 역할
- 충성 수치
- 위험 수치
- 스킬 설명

## 결과

- 주요 인물 5명이 카드형 도감으로 표시된다.
- 전설 등급 `엘리엇 카르테인`은 황금색 테두리와 강한 아우라가 적용된다.
- 희귀/고급 등급도 각각 다른 색상과 아우라를 가진다.
- 모바일에서는 인물 카드가 1열로 정렬된다.
