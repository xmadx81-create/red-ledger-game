## 디자인 방향

- 다크 행정 누아르
- 버건디/블랙/아이보리 계열
- 과장된 공포보다 차가운 행정적 위화감
- 장부, 봉인 문서, 재고표, 보고서 중심의 UI
- 표면 업무와 이면 보고가 겹쳐지는 구조
