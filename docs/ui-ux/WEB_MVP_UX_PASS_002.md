# Web MVP UX 2차 패스 기록

## 기록 일자

2026-06-15

## 작업 목적

플레이 화면의 정보 표시를 정리하고, 선택 기록과 보고서 화면의 가독성을 개선했습니다.

## 적용 파일

```text
src/web-mvp/app.js
src/web-mvp/styles.css
```

## 주요 변경

### 선택 이력 개선

선택 이력을 긴 코드 문자열이 아니라 카드형 기록으로 표시하도록 변경했습니다.

### 사건 카드 개선

사건 카드에 다음 정보를 추가했습니다.

```text
이벤트 ID
사건 유형
위험도
```

### 관련 인물 표시 개선

관련 인물을 칩 형태로 표시하도록 변경했습니다.

### 선택 결과 화면 개선

선택지 코드, 선택지 문구, 결과 요약, 자원 변화, 인물 반응, 관계 변화를 구분했습니다.

### 자원 변화 표시 개선

자원 변화는 증가, 감소, 변화 없음으로 구분해 표시합니다.

### 최종 보고서 개선

최종 보고서의 수급 판정과 선택 이력을 더 읽기 쉽게 정리했습니다.
