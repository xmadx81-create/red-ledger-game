# 이미지 자산 제작 계획 001

## 기록 일자

2026-06-15

## 목적

기존에 생성한 이미지 자산을 폐기하지 않고 활용하며, Web MVP와 Figma 기준에 맞춰 추가 이미지 제작을 진행합니다.

## 기본 원칙

```text
기존 이미지 = 레퍼런스/후보 자산으로 보관
추가 이미지 = 화면 용도별로 계획 생성
즉시 구현 = 필요한 것만 최소 적용
고급 아트 = 화면 구조 안정화 이후 적용
```

## 기존 이미지 활용 방침

기존 생성 이미지는 다음 용도로 분류합니다.

```text
1. 분위기 레퍼런스
2. 배경 후보
3. 문서/장부 텍스처 후보
4. 인물 초상 후보
5. 아이콘/장식 후보
```

바로 Web MVP에 삽입하기보다는, 먼저 Figma의 `07_Visual_References` 또는 GitHub의 `assets/references` 기준으로 정리합니다.

## 추가 이미지 제작 분류

### 1. 타이틀 배경

용도:

```text
SCR-001 타이틀 화면
```

이미지 방향:

```text
어두운 혈연센터 외관
비 오는 밤
붉은 조명
행정기관처럼 차가운 건물
과한 고어 없음
```

### 2. 봉인 문서/장부 텍스처

용도:

```text
보고서 카드
최종 평가 화면
이벤트 문서 배경
```

이미지 방향:

```text
낡은 장부
봉인 도장
혈색 잉크
아이보리 종이
관료적 문서 양식
```

### 3. 주요 인물 초상 후보

대상:

```text
서윤재
김도현
박세연
정하린
엘리엇 카르테인
```

초기에는 완성 일러스트보다 실루엣/반신 콘셉트 후보로 제작합니다.

### 4. 자원 아이콘

대상:

```text
혈액 재고
혈액 수요
가문 자금
인간 신뢰
언론 노출
보안 등급
가문 만족도
조직 불안
암거래 단서
```

아이콘은 작은 크기에서도 식별되어야 하므로 단순한 문장형 심볼을 우선합니다.

### 5. 세력/가문 문장

용도:

```text
카르테인 가문
백십자재단
혈연센터
암거래망
언론 세력
```

디자인 방향:

```text
문장학 느낌
봉인 도장 느낌
검은 금속과 붉은 왁스
기관 로고처럼 보이는 이중성
```

## 우선 제작 순서

```text
1. 타이틀 배경 1종
2. 봉인 장부 텍스처 1종
3. 카르테인 가문 문장 1종
4. 혈연센터 내부 배경 1종
5. 주요 인물 실루엣 5종
6. 자원 아이콘 세트
```

## Web MVP 반영 기준

이미지는 바로 대량 삽입하지 않습니다.

1차 반영은 다음만 적용합니다.

```text
타이틀 배경
문서 카드 텍스처
최종 보고서 배경 장식
```

캐릭터 초상과 아이콘은 UI 구조가 안정된 뒤 적용합니다.
