## 주의 사항

### 1. 자원명 표현

데이터 기준명은 `혈액 재고`, `혈액 수요`입니다.

UI 노출명으로는 `핵심 재고`, `이면 수요`가 쓰일 수 있습니다.

결론:

- 계산식과 Google Sheets 데이터 컬럼은 `혈액 재고`, `혈액 수요`를 사용합니다.
- 화면 문구에서는 장면 분위기에 따라 `핵심 재고`, `이면 수요`를 사용할 수 있습니다.

### 2. 선택지 CSV 분할

선택지 데이터는 두 CSV로 나뉘어 있습니다.

실제 Google Sheets 반영 시에는 두 파일을 하나의 `06_선택지` 탭에 이어 붙입니다.
