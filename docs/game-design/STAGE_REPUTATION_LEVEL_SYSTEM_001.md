# 스테이지 · 명성 · 레벨업 시스템 설계 001

## 기록 일자

2026-06-15

## 설계 목적

`적혈의 장부`는 엔딩형 게임이 아니라 지속 성장형 게임입니다.

헌혈의 집처럼 보이는 혈연센터 컨셉은 세계관의 표면이며, 실제 게임 구조는 다음을 중심으로 확장됩니다.

```text
캐릭터 레벨업
명성 시스템
스테이지 난이도별 클리어
집단 전투
퍼즐
아케이드
전략
운영
```

## 핵심 구조

```text
운영 본부 → 스테이지 선택 → 미션 유형 선택/진입 → 플레이 → 보상/정산 → 성장 → 다음 난이도/지역 개방
```

## 스테이지 구조

스테이지는 하나의 끝이 아니라 반복 도전 가능한 성장 단위입니다.

### 기본 속성

```text
stageId
stageName
region
recommendedLevel
difficulty
missionTypes
clearRank
rewards
unlockConditions
```

### 난이도

```text
Normal
Hard
Elite
Nightmare
Abyss
```

난이도가 오를수록 다음이 강화됩니다.

```text
위험도
적/문제 패턴
제한 조건
보상
명성 증가량
희귀 자산 획득률
```

## 미션 유형

전투만 강제하지 않습니다.

```text
집단 전투
보스전
잠입
퍼즐
아케이드 회피
수송
보급 전략
방어전
조사
협상
해킹/해독
구출
위기 대응
```

각 스테이지는 단일 유형 또는 복합 유형을 가질 수 있습니다.

## 캐릭터 레벨업

캐릭터는 반복 플레이의 핵심 보상입니다.

### 캐릭터 성장 값

```text
level
exp
skillPoint
roleMastery
gearSlots
trait
relationship
awakeningProgress
```

### 역할군

```text
운영형
전투형
의료형
정보형
교섭형
잠입형
물류형
방어형
퍼즐형
아케이드형
```

## 명성 시스템

명성은 단일 점수가 아니라 여러 축으로 나뉩니다.

```text
혈연센터 명성
가문 명성
인간 사회 신뢰
암시장 평판
언론 주목도
조직 위상
지역 영향력
```

명성은 다음에 영향을 줍니다.

```text
스테이지 개방
NPC 반응
상점 품목
동료 영입
보상 배율
위험 이벤트 발생률
세력 외교
```

## 클리어 랭크

기존 엔딩 등급은 클리어 랭크로 변경합니다.

```text
S
A
B
C
D
Failure
```

랭크 산정 기준:

```text
미션 성공 여부
소요 턴/시간
자원 손실
캐릭터 생존/피로
명성 변화
위험 노출
보너스 조건 달성
```

## 중독성 대신 반복 몰입 설계

게임을 끝없이 하게 만드는 핵심은 강제성이 아니라 성장 욕구입니다.

사용할 구조:

```text
다음 난이도 개방
새 캐릭터 영입
새 스킬 해금
명성 등급 상승
지역 장악도 상승
희귀 장비 수집
스테이지 랭크 재도전
조합 실험
시즌 목표
```

피해야 할 구조:

```text
강제 접속 보상
과도한 피로도 압박
불합리한 확률 장벽
과금 없이는 진행 불가한 구조
단순 반복 노가다
```
