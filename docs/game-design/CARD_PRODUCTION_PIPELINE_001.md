# 카드 제작 파이프라인 001

## 목적

이 문서는 `헌혈의 집` 캐릭터 도감 제작 방식을 안정화하기 위한 카드 제작 규칙입니다.

이후 캐릭터 이미지는 다음 원칙을 따른다.

```text
1캐릭터 = 1이미지 = 1도감 카드
```

## 이전 문제

AI가 카드 전체를 한 번에 생성하면 다음 문제가 반복 발생했다.

```text
이름 오기
ID 오기
직업 오기
종족 오기
성향 오기
기획 데이터와 맞지 않는 역할 생성
여러 캐릭터가 한 장에 나오는 시트형 이미지 발생
도감 전체 UI가 이미지로 생성되는 문제
```

따라서 앞으로는 카드 전체 텍스트를 이미지 생성에 맡기지 않는다.

## 새 제작 방식

```text
1. AI는 캐릭터 일러스트 또는 배경 없는 캐릭터 원화만 생성한다.
2. 카드 프레임, 이름, ID, 종족, 직업, 등급, 태그, 아우라는 Web MVP 코드가 출력한다.
3. characterId 기준으로 이미지와 데이터를 연결한다.
4. 도감에는 코드 기반 카드만 사용한다.
```

## 카드 레이아웃 기준

후보 E를 공식 카드 템플릿으로 채택한다.

구성:

```text
상단: 프로젝트 로고/세계관명, ID, 등급 배지
중앙 우측: 캐릭터 일러스트
중앙 후면: 아우라 후광
좌측: 이름, 영문명, 기본 정보
하단: 아우라 설명, 소개, 핵심 태그, 희귀도
```

## 등급별 테두리 규칙

| 등급 | 코드 | 테두리 | 아우라 강도 | 용도 |
|---|---|---|---|---|
| 일반 | N/C | 무채색/회색 | 약함 | 기본 NPC, 초반 영웅 |
| 고급 | R | 초록 | 낮음 | 역할 특화 캐릭터 |
| 희귀 | SR | 파랑 | 중간 | 전투/운영 핵심 캐릭터 |
| 영웅 | EP | 보라 | 강함 | 메인 스토리 주요 캐릭터 |
| 전설 | L | 황금 | 매우 강함 | 시즌 핵심/고대/전설 캐릭터 |

## 성향별 아우라 규칙

| 성향 | 아우라 색 | 의미 |
|---|---|---|
| 선 | 푸른빛 | 공생 질서를 지키는 캐릭터 |
| 악 | 검붉은빛 | 명확한 적대 캐릭터 |
| 중립/모호 | 보라빛 | 회유/배신/동맹 분기 가능 캐릭터 |
| 전설 | 황금빛 | 세계관 핵심 또는 전설급 캐릭터 |

전설급 캐릭터는 기본 성향과 별개로 골드 아우라를 우선 적용한다.

## 이미지 생성 금지 사항

```text
여러 캐릭터 한 장 금지
도감 전체 화면 생성 금지
카드 그리드 생성 금지
시트형 이미지 금지
공식 기관 로고/상표/실제 단체명 사용 금지
기획에 없는 이름/ID/종족/직업을 이미지 안에 직접 생성하지 않기
```

## 이미지 생성 허용 사항

```text
캐릭터 1명 단독 일러스트
상반신 또는 전신
투명/단색/간단한 배경
후광용 아우라 표현
법적 문제 없는 오리지널 헌혈센터풍 의상
도감 카드에 삽입 가능한 세로형 구도
```
