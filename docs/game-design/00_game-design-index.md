# 헌혈의 집 — 게임 기획 인덱스

작성일: 2026-06-15 (장르 전환 반영)

## 1. 프로젝트 개요

`헌혈의 집`은 밝고 명랑한 동네 헌혈센터를 운영하며, 인간과 이종족이 공생하기 위한 **혈맹 카드를 모으고 카드 배틀로 사건을 해결하는 수집형 카드 배틀 RPG**다.

표면은 깨끗한 지역 헌혈센터(봉사·회복·커뮤니티), 이면은 혈액을 둘러싼 공생 협약과 비밀이다. `밝은 표면, 은근한 이면.`

> **장르 전환 안내**: 이 프로젝트는 초기 "운영 시뮬레이션"에서 **수집형 카드 배틀 RPG**로 전환되었다. 최신 기준 문서는 아래 마스터 기획서다.

## 2. 마스터 기획서 (최신 기준)

```text
CARD_GAME_DESIGN_001.md   ← 현재 마스터 기획서 (장르 전환 기준)
```

전투/성장/시너지/스테이지/톤/데이터 매핑은 모두 이 문서를 우선한다.

## 4. 폐기 / 재작성 대상 (옛 시뮬레이션 기준)

아래 문서·개념은 옛 "7일 운영 시뮬레이션 + 엔딩" 기준이라 **폐기 또는 재해석** 대상이다. 내용 참고는 가능하나 최신 기준이 아니다.

| 문서/개념 | 상태 |
|---|---|
| `01_core-loop.md` (7일 운영 루프) | 폐기 → `CORE_LOOP_REDESIGN_001.md` 대체 |
| `05_endings.md` (엔딩 분기) | 폐기 (엔딩 없는 지속형) |
| `MVP_7DAYS_PLAN.md` (7일 MVP) | 재해석 → 카드 전투 MVP로 전환 |
| 가문 자금/만족도/조직 불안 자원 | 배경으로 격하 |
| 제목 "적혈의 장부 / Red Ledger" | 폐기 → "헌혈의 집" |
