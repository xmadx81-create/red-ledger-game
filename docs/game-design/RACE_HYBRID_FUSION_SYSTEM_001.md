# 종족 · 하이브리드 · 합성 가챠 시스템 001

작성일: 2026-06-15
상위 문서: `CARD_GAME_DESIGN_001.md` (마스터 기획서)
문서 성격: 카드 다양성·등급 세분화·합성(가챠)·무료 공급 정형화

> 이 문서는 "**일반 카드가 압도적으로 많은 다양성**", "**모든 종족 쌍의 하이브리드**", "**카드 뽑기 없는 합성 가챠**", "**전설까지 아주 오래 걸리는 세분화 등급**", "**일반 카드 무료 정기 공급**"을 정형화한다.

---

## 1. 설계 철학

```text
1. 다양성의 기반은 "일반(N) 카드의 양"이다. N이 많을수록 도감과 합성 경제가 풍부해진다.
2. 강함은 "좋은 카드를 뽑는 것"이 아니라 "약한 카드를 합성으로 키우는 것"에서 나온다.
3. 뽑기(pull) 가챠는 없다. 두근거림은 "합성 결과"에서 발생한다 = 합성 가챠.
4. 전설은 흔하면 안 된다. 합성 사다리를 길고 세분화해 도달까지 오래 걸리게 한다.
5. 일반 카드는 무료로 꾸준히 제공해, 유저가 항상 합성할 재료를 갖게 한다.
```

`character-production-balance.json`의 1,000장 장기 목표(N 45% / R 30% / SR 17% / EP 7% / L 1%)와 정합한다. 즉 **N이 전체의 절반 가까이**여야 한다.

## 2. 종족 (Base Races)

캐릭터 다양성의 1차 축. 기존 로스터와 사용자 확장 요청을 통합한 정식 종족 목록이다.

| 종족 | 코드 | 분류 | 상태 | 비고 |
|---|---|---|---|---|
| 인간 | HUM | 기본 | 기존 | 현장/구조/방어/체술 |
| 뱀파이어(뱀프) | VMP | 기본 | 기존 | 혈액술/정신계/은폐 |
| 늑대인간 | WRW | 기본 | 기존 | 추적/야간/근육형 |
| 드워프 | DWF | 기본 | 기존 | 공학/방어 설비 |
| 요정·엘프 | ELF | 기본 | 기존 | 자연/치유/룬 |
| 정령 | SPR | 기본 | 기존 | 원소/회복/수송 지원 |
| 호빗(코빗) | HOB | 기본 | 기존 | 배달/정보/회복 아이템 |
| 골렘 | GLM | 기본 | 기존 | 석재/금속/거대 방어 |
| 위저드 | WIZ | 기본 | 신규 | ※ 직군 성격 있으나 종족으로 포함(추후 재검토) |
| 악마 | DEM | 기본 | 신규 | 갈등/유혹/계약 — EVIL 보강원 |
| 천사 | ANG | 기본 | 신규 | 가호/심판/정화 |
| 수인 | BST | 기본 | 신규 | 야성/감각/기동 |
| 도깨비 | DKB | 기본 | 신규 | 한국 설화, 장난/방망이/변신 |
| 요괴(신화종) | YOK | 기본 | 신규 | 손오공류 — 신화 영웅·신수·여의봉형 |

- 표기: 데이터상 종족명은 "뱀프", 표시명은 "뱀파이어"로 둘 수 있다. 코드 우선.
- **고대/초월 변종**(초고대뱀프 `AVP`, 초고대늑대인간 `AWR` 등)은 기본 종족이 아니라 **상위 등급(전설급) 변종 라인**으로 취급한다.
- 위저드는 "직군 vs 종족" 논쟁이 있어 플래그. 일단 종족 슬롯으로 두되, 추후 직업으로 강등 가능.

## 3. 범용 하이브리드 (Universal Hybrid)

다양성의 폭발 지점. **하이브리드는 특정 종족 전용이 아니라, 모든 종족 쌍에서 발생한다.**

### 3.1 원칙
```text
- 기본 종족 2종을 부모로 갖는 모든 조합이 하이브리드로 존재한다.
- 예: 인간×뱀파이어, 늑대인간×인간, 늑대인간×뱀파이어, 드워프×골렘, 천사×악마 ...
- 하이브리드는 두 부모 종족의 특성(아우라/직업 풀/시너지)을 부분 계승한다.
```

### 3.2 규모
- 기본 종족 14종 → 2종 하이브리드 조합 = C(14,2) = **91종**.
- 기본 14 + 하이브리드 91 = **105개 종족 라인**. 각 라인 안에서 직업·연령·성향·실루엣 변주 → 도감 1,000장+ 다양성을 자연스럽게 채운다.

### 3.3 식별 규칙(안)
```text
하이브리드 종족코드: HYB-{부모A}-{부모B}  (알파벳 정렬로 중복 방지)
  예) HYB-HUM-VMP (인간×뱀파이어), HYB-VMP-WRW (뱀파이어×늑대인간)
캐릭터/카드 ID·cardUid는 기존 규칙 유지 (CARD-{CHARACTER_ID}-{RARITY}-{SERIAL}).
```

### 3.4 특성 계승(안)
| 항목 | 계승 방식 |
|---|---|
| 아우라 | 두 부모 아우라의 혼합/이중 아우라 (불안정 연출 가능) |
| 직업 풀 | 두 부모 직업군에서 교차 선택 |
| 시너지 | 두 부모 종족 시너지를 "다리"처럼 양쪽에 일부 기여 |
| 비주얼 | `CHARACTER_VISUAL_DIVERSITY_GUIDE_001.md` — 부모 실루엣을 혼합하되 반복 금지 |

> 톤 주의: 하이브리드의 "불안정/이중성"을 다크하게 과장하지 않는다. 기본 도감 톤은 밝게 유지(`VISUAL_TONE_GUIDE_001.md`).

## 4. 세분화 등급 시스템

### 4.1 메이저 등급 (5단계, 유지)
`card-style-system.json` 그대로: N → R → SR → EP → L.

| 등급 | 코드 | 의미 |
|---|---|---|
| 일반 | N | 도감 기반층, 합성 연료, 압도적 다수 |
| 고급 | R | 역할 특화 |
| 희귀 | SR | 전투/운영 핵심 |
| 영웅 | EP | 메인 스토리급 |
| 전설 | L | 시즌·세계관 핵심 (극소수) |

### 4.2 세분화 — 등급 안의 단계(★)
각 메이저 등급은 내부에 **다단계 강화 스테이지**를 가진다. (장비 `item-grades.json`의 등급별 강화 레벨 모델과 동일 사상)

```text
N: ★1 → ★2 → ★3 → ★4 → ★5
R: ★1 → ... → ★6
SR: ★1 → ... → ★7
EP: ★1 → ... → ★8
L: ★1 → ... → ★10
```
- 같은 등급 내 ★ 상승은 **강화(Enhance)**로 진행(상대적으로 안정적).
- **다음 메이저 등급으로의 도약(N→R, R→SR ...)은 "합성(Fusion)" 판정**으로만 일어나며 확률이 낮다.

### 4.3 전설까지의 거리(장기화)
```text
N★max → [합성] → R → ... R★max → [합성] → SR → ... → L
메이저 도약 합성은 위로 갈수록 성공률이 급감한다.
따라서 전설(L) 도달은 누적 합성 시도·재료·시간이 크게 요구된다(의도된 장기 목표).
```

## 5. 합성 가챠 (Fusion Gacha)

### 5.1 핵심
```text
카드 뽑기(pull)는 없다.
합성 = 카드/재료를 합쳐 상위 결과를 "확률+행운"으로 노리는 행위.
대부분 실패하고, 아주 가끔 상위 단계/상위 카드가 나온다 → 이것이 이 게임의 가챠다.
```

### 5.2 합성 입력
| 입력 | 예 |
|---|---|
| 베이스 카드 | 키우려는 카드 |
| 재료 카드 | 중복 카드·하위 등급 카드 |
| 합성석/촉매 | 성공률·행운 보정 아이템 |
| 비용 | 혈액/재화 |

### 5.3 판정식(설계 방향, 수치는 별도 밸런스)
```text
성공확률 = 기본확률(등급 도약별 고정, 위로 갈수록 낮음)
         + 재료 보정(품질·동일 종족/성향 보너스)
         + 행운 보정(캐릭터/유저 행운 수치, 이벤트)
         + 누적 보정(연속 실패 시 상승하는 천장형 보정)
```

### 5.4 결과 처리 (윤리 가드레일)
`CORE_LOOP_REDESIGN_001.md`의 "유저를 괴롭히지 않는다" 원칙을 합성에도 적용한다.

```text
성공: 상위 ★ 또는 상위 등급 카드 생성. 상위 등급 신규 카드는 새 cardUid 발급.
실패: 재료는 소모되나
   - 누적 보정(천장)이 올라가 다음 성공 확률이 높아진다.
   - 소량 부산물(합성 분진/경험치)을 지급해 완전 손실을 막는다.
공개: 합성 확률·천장 수치는 투명하게 표기한다(확률 조작 금지).
금지: 실패만 반복시켜 과금/노가다를 강요하는 설계.
```

> 즉 "대부분 실패"는 유지하되, **천장형 누적 보정**으로 "꾸준히 하면 언젠가 도약한다"를 보장해 무한 성장 동기를 만든다.

## 6. 카드 공급 모델 (뽑기 없음)

### 6.1 일반(N) 카드 무료 정기 공급
```text
- 출석/로그인 보상: 연속 접속 시 N 카드 지급 (예: 7일 연속 → +1장 또는 +5장).
- 데일리 미션·스테이지 드랍으로도 N 카드 상시 확보.
- 목적: 유저가 "항상 합성할 재료"를 갖게 해 성장 액션이 끊기지 않게 한다.
```

### 6.2 상위 카드 획득 경로
```text
- 직접 뽑기 없음.
- 상위 등급(R 이상)은 원칙적으로 "합성 도약"으로 만든다.
- 일부 EP/L은 스토리·시즌·고난도 던전 보상으로 한정 지급(합성 외 경로).
```

### 6.3 과금(정책 방향, 확정 아님)
```text
- 뽑기 상품 없음.
- 과금은 (도입한다면) 합성 재료/편의/시즌 패스 중심.
- 확률 조작·뽑기 압박·피로도 강매 금지.
```

## 7. 다양성 × 합성 경제의 연결

```text
다양한 N 카드 대량 공급(종족 14 × 하이브리드 91 × 변주)
   → 유저가 합성 재료를 풍부하게 보유
      → 합성 가챠로 상위 등급 도전(대부분 실패, 천장으로 보장된 진전)
         → 드물게 상위 카드 탄생(새 cardUid) = 강한 성취감
            → 더 높은 난이도/시즌 콘텐츠 도전
               → 더 좋은 재료·한정 카드 확보 → (반복, 종료 없음)
```
