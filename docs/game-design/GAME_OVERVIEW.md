
## 한 줄 콘셉트

`적혈의 장부`는 헌혈센터처럼 보이는 혈연센터를 운영하며, 그 이면에서 뱀파이어 가문의 비밀 혈액 수급망을 관리하는 **다크 행정 누아르 운영 시뮬레이션** 게임입니다.

플레이어는 비밀이 들켜서도 안 되고, 혈액 공급이 수요를 충족하지 못해 뱀파이어들이 통제를 잃고 날뛰게 해서도 안 됩니다. 과잉 은폐와 과잉 공급 사이에서 중용을 지키며, 인간들끼리 혈액을 몰래 사고파는 암거래 흐름까지 찾아내고 통제해야 합니다.

## 핵심 요약

플레이어는 백십자재단 산하 `혈연센터`의 실무 책임자가 되어 겉으로는 정상적인 공익 의료 행정 조직을 운영하고, 뒤로는 `카르테인 가문 혈액관리국`의 지시를 받아 혈액 재고, 인간 사회의 신뢰, 언론 노출, 내부 조직 불안, 가문 수요, 암거래 단서를 동시에 관리해야 합니다.

겉으로는 평범한 보고서, 재고표, 민원, 일정, 직원 관리처럼 보이지만, 플레이어의 모든 결정은 뱀파이어 가문의 생존과 인간 사회에 대한 은폐 전략에 영향을 줍니다.

이 게임의 핵심은 단순히 혈액을 많이 모으는 것이 아닙니다. 너무 적게 공급하면 가문 내부의 갈증과 폭주 위험이 커지고, 너무 무리하게 확보하면 인간 사회의 의심과 암시장 교란이 커집니다. 플레이어는 항상 `들키지 않을 정도로만`, `부족하지 않을 만큼만`, `무너지지 않을 수준으로만` 움직여야 합니다.

## 핵심 딜레마

게임의 중심 딜레마는 다음 네 가지입니다.

```text
비밀이 드러나면 조직이 무너진다.
공급이 부족하면 뱀파이어들이 통제를 잃는다.
과잉 대응하면 인간 사회의 의심이 커진다.
암거래를 방치하면 혈액 수급망의 통제권을 잃는다.
```

플레이어는 이 네 가지 위험 사이에서 중용을 지켜야 합니다.

## 장르

```text
경영 시뮬레이션

조직관리 시뮬레이션
다크 행정 누아르
은폐/수급 균형 관리
```

## 플레이어 역할

플레이어는 `서윤재 실장` 또는 그에 준하는 총괄 관리자 역할을 맡습니다.

플레이어의 임무는 다음과 같습니다.

- 혈연센터의 정상 운영 유지
- 혈액 재고 확보와 배분 관리
- 가문 수요와 실제 공급량의 균형 유지
- 카르테인 가문의 요구 대응
- 인간 사회의 신뢰 유지
- 언론과 민원의 의심 차단
- 인간 사회 내부의 불법 혈액 암거래 단서 탐지
- 암거래가 가문 수급망을 교란하지 않도록 통제
- 내부 조직의 불안과 배신 가능성 관리
- 7일 동안 조직을 붕괴시키지 않고 운영 결과를 보고

## 세계관 기본 구조

### 표면 조직

```text
백십자재단 산하 혈연센터
```

표면적으로는 헌혈, 혈액 보관, 공익 캠페인, 의료 지원을 담당하는 민간 재단 조직입니다.

이 조직은 인간 사회에서 합법적이고 공익적인 이미지를 유지해야 하며, 시민 신뢰와 행정적 투명성을 가장 중요한 외부 자산으로 삼습니다.

### 이면 조직

```text
카르테인 가문 혈액관리국
```

이면에서는 오래된 뱀파이어 가문인 카르테인 가문이 혈액 수급망을 통제합니다.

카르테인 가문은 불멸성, 자본, 정치적 영향력, 오래된 계약망을 통해 인간 사회의 일부 조직을 우회적으로 지배하고 있습니다.

### 제3의 위험: 인간 암거래망

혈연센터와 가문이 통제하는 공식/비공식 수급망 외부에는 인간들끼리 혈액을 몰래 사고파는 암거래 흐름이 존재합니다.

이 암거래망은 처음에는 작은 이상 징후처럼 보입니다.

- 혈액 재고 숫자의 미세한 불일치
- 특정 지역에서 반복되는 이상 주문
- 기록상 폐기 처리된 혈액의 재등장
- 익명 제보와 민원
- 하부 직원의 수상한 접촉
- 기자가 추적하는 비공식 거래 소문

플레이어는 이 암거래망을 무조건 공개적으로 단속할 수 없습니다. 공개 대응은 혈연센터의 비밀을 드러낼 수 있고, 방치는 가문의 수급망을 흔들 수 있습니다. 따라서 플레이어는 조용히 단서를 모으고, 필요할 때 회수·차단·거래·은폐 중 하나를 선택해야 합니다.

### 플레이 공간

1차 MVP의 주요 공간은 하나의 혈연센터와 그 주변 조직입니다.

- 센터 사무실
- 혈액 보관실
- 야간 진료실
- 물류 연결망
- 언론 대응 창구
- 가문 보고 라인
- 내부 직원 관리 영역
- 암거래 단서 추적 라인

## 핵심 플레이 경험

플레이어는 매일 제한된 정보와 자원을 바탕으로 결정을 내려야 합니다.

핵심 재미는 다음 긴장감에서 발생합니다.

```text
공익 행정처럼 보이는 업무
vs
비밀 혈액 수급망을 유지해야 하는 어두운 목적
```

```text
인간 사회의 신뢰 유지
vs
가문이 요구하는 혈액 공급량 충족
```

```text
공급 부족으로 인한 뱀파이어 폭주 위험
vs
과잉 확보로 인한 인간 사회의 의심 증가
```

```text
암거래망을 이용해 부족분을 메울 유혹
vs
암거래망 때문에 통제권을 잃을 위험
```

```text
단기 위기 해결
vs
장기적으로 더 큰 폭로와 붕괴 위험 증가
```

## 중용 시스템의 방향

이 게임은 단순히 높은 수치를 만드는 게임이 아닙니다.

핵심은 적정선을 유지하는 것입니다.

| 축 | 너무 낮을 때 | 너무 높을 때 |
|---|---|---|
| 혈액 재고 | 가문 불만, 뱀파이어 폭주 위험 | 보관/폐기 문제, 기록 불일치 위험 |
| 인간 신뢰 | 민원, 조사, 언론 관심 증가 | 과도한 공익 이미지로 외부 접촉 증가 |
| 언론 노출 | 낮을수록 안정 | 높으면 폭로와 조사 위험 |
| 조직 불안 | 낮을수록 안정 | 높으면 내부 배신과 실수 증가 |
| 가문 만족도 | 낮으면 강제 개입 | 높아도 무리한 추가 요구 가능 |
| 암거래 단서 | 낮으면 감시 실패 가능 | 높으면 직접 충돌과 노출 위험 증가 |

중용 시스템은 플레이어가 `최대화`보다 `균형 유지`에 집중하도록 설계합니다.

## 핵심 자원

1차 MVP에서 관리할 핵심 자원 후보는 다음과 같습니다.

| 자원 | 의미 |
|---|---|
| 혈액 재고 | 센터가 보유한 혈액량. 부족하면 가문 불만과 폭주 위험이 증가합니다. |
| 혈액 수요 | 가문과 개별 뱀파이어들이 요구하는 혈액량입니다. 공급과 균형이 맞지 않으면 위기가 발생합니다. |
| 가문 자금 | 은폐, 매수, 설비, 인력 관리에 쓰이는 자금입니다. |
| 가문 위신 | 카르테인 가문 내부에서 플레이어 조직이 받는 평가입니다. |
| 인간 신뢰 | 인간 사회가 혈연센터를 신뢰하는 정도입니다. |
| 언론 노출 | 기자, 제보, 의혹 보도로 인해 외부에 드러나는 위험도입니다. |
| 조직 불안 | 내부 직원, 하부 조직, 협력자의 동요 정도입니다. |
| 보안 등급 | 비밀 유지 체계의 안정성입니다. |
| 가문 만족도 | 원로와 핵심 뱀파이어 세력이 현재 운영에 만족하는 정도입니다. |
| 암거래 단서 | 인간 사회 내부의 불법 혈액 거래 흐름을 파악한 정도입니다. |

## 기본 게임 루프

1차 MVP의 기본 루프는 하루 단위로 진행됩니다.

```text
Day 시작 보고
→ 현재 자원 상태 확인
→ 혈액 수요와 공급량 비교
→ 오늘의 사건 발생
→ 암거래/언론/가문/내부 조직 단서 확인
→ 선택지 결정
→ 자원 변화 반영
→ 내부/외부 반응 확인
→ 추가 사건 또는 운영 결정
→ Day 종료 보고
→ 다음 Day 진행
```

## 1차 MVP 범위

1차 MVP는 `7일 운영 구조`를 기준으로 합니다.

### MVP 목표

- 핵심 세계관 전달
- 하루 단위 운영 루프 검증
- 자원 변화 시스템 검증
- 공급과 수요의 균형 시스템 검증
- 중용 시스템의 재미 검증
- 암거래 단서 추적 구조 검증
- 선택지 결과 시스템 검증
- 주요 인물의 역할과 갈등 구조 검증
- 7일 결과에 따른 간단한 엔딩 분기 검증

### MVP에 포함할 것

- Day 1~Day 7 진행 구조
- 핵심 자원과 위험 지표
- 혈액 공급/수요 균형 판단
- 인간 암거래 단서 추적 이벤트
- 주요 인물 5명
- 일차별 주요 사건
- 사건별 선택지
- 선택 결과에 따른 자원 변화
- 최종 평가 또는 엔딩 분기

### MVP에서 제외할 것

- 대규모 지역 확장
- 복잡한 전투 시스템
- 실시간 액션 조작
- 온라인 멀티플레이
- 실제 기관명 사용
- 현실의 불법 거래 방법을 구체적으로 설명하는 구조
- 과도하게 복잡한 경제 시뮬레이션

## 시각 방향

초기 시각 방향은 다음을 권장합니다.

- 어두운 배경
- 혈액을 직접적으로 과장하기보다 문서, 냉장 보관, 보고서, 도장, 파일철 이미지 활용
- 붉은색은 강조색으로 제한 사용
- 공공기관/재단 느낌의 차분한 UI
- 사건이 심각해질수록 화면의 질서가 무너지는 연출
- 수요와 공급이 어긋날수록 보고서 숫자가 불안정하게 보이는 연출
- 암거래 단서는 처음에는 작은 오류, 누락, 중복 기록처럼 표현

## 플레이 목표

플레이어의 단순 목표는 7일 동안 센터를 무사히 운영하는 것입니다.

그러나 실제 목표는 다음 균형을 유지하는 것입니다.

```text
혈액을 충분히 확보한다.
그러나 과잉 확보로 의심을 사지 않는다.
가문을 만족시킨다.
그러나 가문의 폭주를 부추기지 않는다.
인간 사회의 의심을 피한다.
그러나 암거래를 방치하지 않는다.
조직 내부의 붕괴를 막는다.
```

## 실패 조건 후보

1차 MVP에서 고려할 실패 조건은 다음과 같습니다.

- 혈액 재고가 0에 도달
- 혈액 수요를 반복적으로 충족하지 못해 뱀파이어 폭주 이벤트 발생
- 언론 노출이 최대치에 도달
- 조직 불안이 최대치에 도달
- 가문 만족도가 0에 도달
- 인간 신뢰가 0에 도달
- 보안 등급이 임계치 이하로 하락
- 암거래망이 통제 불가능한 수준으로 성장
- 암거래 단속이 과도해 혈연센터의 비밀이 역으로 노출

## 엔딩 방향 후보

1차 MVP에서는 복잡한 엔딩보다 운영 결과에 따른 간단한 평가형 엔딩을 우선합니다.

예시:

| 엔딩 유형 | 조건 방향 |
|---|---|
| 안정 운영 | 수요와 공급 균형이 좋고 큰 폭로 없이 7일 종료 |
| 가문 우선 | 가문 만족도는 높지만 인간 신뢰와 보안 안정성이 낮음 |
| 인간 사회 우선 | 인간 신뢰는 높지만 가문 만족도와 공급 안정성이 낮음 |
| 은폐 실패 | 언론 노출 또는 보안 실패로 비밀이 드러남 |
| 공급 붕괴 | 혈액 수요를 충족하지 못해 뱀파이어 통제 실패 |
| 암거래 잠식 | 인간 암거래망이 커져 수급망 통제권을 상실 |
| 내부 붕괴 | 조직 불안이 커져 내부 배신 또는 사고 발생 |

## 설계 원칙

- 실제 기관명은 사용하지 않고 가상 조직명을 사용합니다.
- 현실의 헌혈 제도를 직접 비방하거나 특정 기관을 지칭하지 않습니다.
- 현실의 불법 거래 방법을 구체적으로 안내하지 않습니다.
- 공포의 핵심은 피 자체보다 행정 시스템 뒤에 숨은 비밀 구조에서 나와야 합니다.
- 선택지는 명확한 정답이 아니라 비용과 대가가 있는 판단이어야 합니다.
- 플레이어는 선악이 아니라 위험관리, 균형, 생존전략 사이에서 고민하게 만듭니다.
- 시스템의 핵심은 최대화가 아니라 중용입니다.
