# Web MVP 컨텐츠 확장 계획 001

## 기록 일자

2026-06-15

## 현재 상태

Web MVP는 7일 운영, 선택지 처리, 자원 변화, 최종 보고서까지 플레이 가능한 상태입니다.

현재는 기능 검증과 기본 UI 패스가 끝난 단계이며, 다음부터는 플레이 판단의 밀도를 높이는 작업이 필요합니다.

## 확장 목표

```text
선택 전 판단 정보 강화
선택 후 결과 해석 강화
최종 보고서의 서사성 강화
반복 회차 플레이 명분 확보
```

## 1차 컨텐츠 확장 항목

### 1. 선택지 영향 미리보기

선택지를 누르기 전에 플레이어가 대략적인 위험을 읽을 수 있어야 합니다.

예시 표시:

```text
주요 영향: 재고, 신뢰, 노출
위험 성격: 은폐 / 가문 우선 / 인간 우선 / 암거래
```

정확한 수치를 모두 노출하기보다는, 초반에는 영향 방향과 성격만 보여주는 것이 좋습니다.

### 2. 최종 보고서 엔딩 문장 강화

현재 최종 보고서는 결과 구조는 있으나, 엔딩 문장의 서사 밀도가 낮습니다.

엔딩 플래그별 문장을 추가합니다.

```text
SUPPLY_STABLE
HUMAN_TRUST
FAMILY_DOMINANT
BLACK_MARKET_DOMINANT
EXPOSED_MEDIA
INTERNAL_COLLAPSE
SUPPLY_COLLAPSE
```

### 3. 사건 문서 분류명 강화

현재 사건 유형은 간단한 단어 중심입니다.

더 문서형으로 바꿀 수 있습니다.

```text
운영 보고
가문 지시
언론 질의
운송 이상
위기 기록
최종 평가
```

### 4. 인물 반응 강화

현재 인물 반응은 한 문장입니다.

향후에는 다음 구조로 확장합니다.

```text
즉시 반응
관계 변화
다음 사건 암시
```

## 우선 적용 순서

```text
1. 선택지 영향 미리보기
2. 최종 보고서 엔딩 문장
3. 사건 카드 문서 분류명
4. 인물 반응 세분화
```

## 주의 사항

컨텐츠 확장보다 먼저 복잡한 일러스트나 고급 연출을 넣지 않습니다.

먼저 텍스트와 정보 구조를 안정화한 뒤, 그 구조에 맞춰 비주얼을 입히는 순서가 적합합니다.
