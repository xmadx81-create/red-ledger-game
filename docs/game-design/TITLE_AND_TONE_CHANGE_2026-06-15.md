# 게임 제목 및 톤 변경 기록

## 기록 일자

2026-06-15

## 변경 요청

게임 제목을 `헌혈의 집`으로 변경합니다.

전체 톤은 어둡고 누아르한 분위기에서 벗어나, 밝고 깨끗한 공공 헌혈센터 느낌으로 전환합니다.

## 변경 기준

```text
게임 표시 제목: 헌혈의 집
UI 톤: 밝음, 깨끗함, 신뢰감, 공공 서비스 느낌
표면 컨셉: 헌혈센터 운영
이면 구조: 비밀 유지, 인간과의 공생, 성장형 전략 RPG
```

## 주의 사항

실제 기관과 혼동되지 않도록 공식 로고, 실제 기관 상징, 실제 홍보 문구는 사용하지 않습니다.

시각적 방향만 다음처럼 차용합니다.

```text
밝은 흰색 배경
깨끗한 카드 UI
붉은 포인트 컬러
파란 보조 컬러
의료/공공 서비스 느낌의 안정감
```

## 기존 톤과의 차이

기존:

```text
다크 행정 누아르
어두운 붉은 배경
비밀 문서 분위기
```

변경:

```text
밝은 헌혈센터 UI
신뢰감 있는 공공 서비스 화면
겉으로는 깨끗하지만 이면에 비밀이 있는 구조
```

## 적용 대상

```text
Web MVP 타이틀
타이틀 화면 문구
색상 토큰
카드/버튼/자원 UI
Figma 화면 톤
이미지 자산 방향
```
