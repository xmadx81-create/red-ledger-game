# 비주얼 톤 가이드 001

## 목적

`헌혈의 집` 프로젝트의 기본 비주얼 톤을 고정한다.

최근 캐릭터 생성 과정에서 다크 판타지, 폐허, 전쟁, 고딕 뱀프, 흑화 분위기가 과도하게 반복되었다. 이는 현재 게임이 목표로 하는 밝고 명랑한 헌혈센터 운영 분위기와 맞지 않는다.

따라서 앞으로 모든 캐릭터, 카드, UI, 배경, 도감 이미지는 아래 톤을 우선한다.

---

## 공식 기본 톤

```text
밝고 명랑한 헌혈의 집
쾌활하고 따뜻한 지역 커뮤니티
깨끗하고 안전한 센터 공간
봉사, 도움, 회복, 일상성
친근한 캐릭터 도감
가볍게 접근 가능한 운영 시뮬레이션
```

핵심 키워드:

```text
밝음
명랑함
쾌활함
따뜻함
친근함
안전함
깔끔함
희망적
커뮤니티
봉사
회복
```

---

## 숨은 설정의 처리 방식

세계관 안에는 뱀프, 비밀 혈액 수급망, 갈등, 음모, 전투 요소가 존재할 수 있다.

하지만 이것은 기본 화면 톤을 어둡게 만드는 이유가 아니다.

```text
표면 톤: 밝고 쾌활한 헌혈센터 운영
이면 톤: 선택지, 스토리, 야간 이벤트에서만 은근히 드러나는 긴장감
```

즉, 어두운 요소는 다음처럼 제한한다.

```text
어두운 배경은 이벤트/던전/야간 장면에만 사용
기본 캐릭터 카드와 도감은 밝은 톤 유지
뱀프 캐릭터도 모두 검은 고딕 귀족처럼 만들지 않음
전투 캐릭터도 폐허/전쟁터보다 센터 주변 사건 해결 느낌 우선
```

---

## 권장 색상 방향

| 용도 | 색상 방향 |
|---|---|
| 기본 UI | 화이트, 크림, 연한 하늘색, 부드러운 코랄 |
| 헌혈센터 공간 | 밝은 실내 조명, 깨끗한 흰색, 따뜻한 붉은 포인트 |
| 선 성향 | 밝은 푸른색, 맑은 청록색 |
| 중립 성향 | 부드러운 보라, 라벤더, 파스텔 보조광 |
| 악 성향 | 너무 어둡지 않은 경고색, 버건디, 딥 레드 포인트 |
| 전설 | 금색이지만 과도한 암흑 배경 금지 |

## 금지 톤

기본 캐릭터 제작에서 아래 톤을 반복 사용하지 않는다.

```text
폐허 도시
전쟁터
고어/피범벅
검은 고딕 성
악마성
어두운 마법진 중심 구도
흑화된 뱀프 귀족 반복
검붉은 화면 전체 장악
공포 게임 포스터 같은 연출
잔혹한 실험실
과도하게 차가운 시체보관소 톤
```

특히 다음 조합은 금지한다.

```text
뱀프 + 검은 고딕복 + 보라 마법진 + 어두운 실험실
전투원 + 폐허 도시 + 폭발 + 전쟁터
헌혈센터 캐릭터 + 피범벅 실험복
```

## 캐릭터 생성 기준

캐릭터 이미지는 다음을 우선한다.

```text
밝은 센터 로비
깨끗한 혈액 보관실
친근한 상담 공간
활기 있는 봉사 현장
따뜻한 지역 거리
명랑한 스태프룸
안전하게 정돈된 연구실
```

어두운 설정을 가진 캐릭터도 다음처럼 처리한다.

```text
검은 배경 대신 밝은 실내 + 은근한 그림자
피 대신 빨간 샘플 튜브나 상징적 패턴
공포 대신 미스터리
잔혹함 대신 긴장감
전쟁 대신 구조/수습/보호 상황
```

## Batch 001 교정 사항

Batch 001에서 하린, 그라프, 태오는 어두운 장면으로 흐르는 경향이 있었다.

향후 조정:

```text
하린: 폐허 방호가 아니라 밝은 센터 입구 대피 유도 톤
그라프: 시체보관소가 아니라 깨끗한 냉장 보관실/기록실 톤
태오: 전쟁터가 아니라 야간 수송 훈련 또는 센터 옥상 구조 지원 톤
```

---

## 이미지 생성 전 톤 체크

이미지를 만들기 전 반드시 확인한다.

```text
1. 이 이미지가 헌혈의 집 기본 톤인 밝고 명랑한 분위기를 해치지 않는가?
2. 어두운 설정이 필요하더라도 화면 전체를 흑화시키지 않았는가?
3. 캐릭터가 도감에서 친근하게 보이는가?
4. 폐허/전쟁/고어/공포 표현이 들어가지 않았는가?
5. 숨은 긴장감은 배경의 작은 요소로만 들어갔는가?
```

```text
밝은 표면, 은근한 이면.
명랑한 헌혈센터 운영이 먼저이고, 다크 판타지는 과하지 않게 숨긴다.
```
