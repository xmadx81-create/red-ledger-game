# 적혈의 장부 — 핵심 자원 설계

작성일: 2026-06-16

## 1. 자원 설계 목적

자원은 플레이어의 선택 결과를 즉시 보여주는 핵심 지표다.

MVP에서는 8개의 자원을 사용한다.

| 자원ID | 자원명 | 초기값 | 범위 | 설계 목적 |
|---|---|---:|---|---|
| RES_BLOOD | 혈액 재고 | 72 | 0~100 | 운영 생존 자원 |
| RES_FUNDS | 가문 자금 | 420 | 0~999 | 선택지 비용과 보강 비용 |
| RES_PRESTIGE | 가문 위신 | 55 | 0~100 | 가문 평가와 상위 엔딩 조건 |
| RES_TRUST | 인간 신뢰 | 61 | 0~100 | 표면 조직 유지 지표 |
| RES_EXPOSURE | 언론 노출 | 18 | 0~100 | 발각 위험 지표, 낮을수록 좋음 |
| RES_UNREST | 조직 불안 | 24 | 0~100 | 내부 배신/실수 위험 지표 |
| RES_SECURITY | 보안 등급 | 2 | 1~5 | 위험 회피와 고급 이벤트 조건 |
| RES_FAMILY | 가문 만족도 | 50 | 0~100 | 7일차 최종 평가 지표 |

## 2. 자원 간 긴장 구조

### 혈액 재고 vs 언론 노출

혈액 재고를 빠르게 확보하면 언론 노출과 조직 불안이 증가할 수 있다.

### 인간 신뢰 vs 가문 만족도

표면 조직을 깨끗하게 운영하면 인간 신뢰는 오르지만, 가문 만족도는 낮아질 수 있다.

### 자금 vs 보안

보안을 강화하면 위험은 줄지만 자금이 줄어든다.

### 조직 불안 vs 운영 효율

강압적 선택은 단기 효율을 높일 수 있지만, 장기적으로 내부 붕괴 엔딩 위험을 높인다.

## 3. MVP 밸런스 기준

MVP에서는 모든 자원이 한 번에 무너지지 않도록 한다.

권장 변화폭:

| 변화 유형 | 권장 수치 |
|---|---:|
| 작은 변화 | ±2~4 |
| 일반 변화 | ±5~8 |
| 큰 변화 | ±10~15 |
| 치명 변화 | ±20 이상 |

## 4. UI 표시 방식

| 자원 | UI 표시 |
|---|---|
| 혈액 재고 | 상단 핵심 카드 |
| 가문 자금 | 상단 핵심 카드 |
| 인간 신뢰 | 상단 핵심 카드 |
| 언론 노출 | 위험도 게이지 |
| 가문 위신 | 가문 패널 |
| 조직 불안 | 내부 관리 카드 |
| 보안 등급 | 등급 배지 |
| 가문 만족도 | 최종 평가 패널 |

## 5. 다음 조정 필요사항

- 자금 단위를 K, 만, 혹은 추상 포인트 중 하나로 확정해야 한다.
- 보안 등급은 1~5 정수형으로 유지한다.
- 언론 노출과 조직 불안은 높을수록 위험한 역방향 자원이다.
