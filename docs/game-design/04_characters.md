# 적혈의 장부 — 캐릭터 설계

작성일: 2026-06-16

## 1. 캐릭터 설계 목적

캐릭터는 단순한 대화 NPC가 아니라, 운영 선택의 압박과 세계관의 긴장을 만들어내는 장치다.

MVP에서는 5명의 핵심 인물을 사용한다.

| 캐릭터ID | 이름 | 소속 | 역할 | 등급 |
|---|---|---|---|---|
| CHR_SEO | 서윤재 | 백십자재단 / 혈연센터 | 총괄 관리자 / 플레이어 대리인 | 희귀 |
| CHR_KIM | 김도현 | 야간 진료실 | 야간 의사 / 기록 관리자 | 고급 |
| CHR_PARK | 박세연 | 외부 물류망 | 물류 협력자 | 고급 |
| CHR_JUNG | 정하린 | 외부 언론 | 기자 / 외부 위협 | 희귀 |
| CHR_ELIOT | 엘리엇 카르테인 | 카르테인 가문 | 가문 원로 / 평가자 | 전설 |

## 2. 인물별 역할

### 서윤재

역할:

- 플레이어의 실무 대리인
- 운영 판단, 결재, 내부 통제 담당

능력:

- 행정 통제
- 위기 판단

스킬 초안:

```text
긴급 결재
→ 자금을 소모해 언론 노출을 감소시킨다.
```

### 김도현

역할:

- 야간 의사
- 기록 관리자
- 혈액 오염/폐기 판단 담당

능력:

- 의료 기록 정리
- 오염 판별

스킬 초안:

```text
무균 판정
→ 폐기 손실을 줄이고 혈액 재고 감소를 완화한다.
```

### 박세연

역할:

- 외부 물류 협력자
- 은밀한 운송 루트 담당

능력:

- 은밀한 수송
- 긴급 회수

스킬 초안:

```text
야간 루트
→ 혈액 재고를 증가시키지만 언론 노출이 소폭 증가한다.
```

### 정하린

역할:

- 외부 기자
- 발각 위험을 만드는 적대 NPC

능력:

- 취재 압박
- 제보 연결

스킬 초안:

```text
추적 기사
→ 언론 노출을 급격히 증가시킨다.
```

### 엘리엇 카르테인

역할:

- 가문 원로
- 최종 평가자
- 가문 만족도와 위신 압박을 만드는 인물

능력:

- 가문 명령
- 최종 평가

스킬 초안:

```text
원로의 요구
→ 가문 만족도가 크게 변동한다.
```

## 3. 캐릭터 수치

| 캐릭터 | 충성 | 위험 | 해석 |
|---|---:|---:|---|
| 서윤재 | 70 | 35 | 안정적인 내부 관리자 |
| 김도현 | 62 | 45 | 실무 능력은 높으나 기록 문제와 연결됨 |
| 박세연 | 55 | 50 | 보상과 압박에 민감한 협력자 |
| 정하린 | 20 | 80 | 낮은 충성, 높은 외부 위험 |
| 엘리엇 카르테인 | 90 | 70 | 가문에는 충성하지만 플레이어에게 압박을 줌 |

## 4. 관계 구조

```text
서윤재
├─ 김도현: 업무 신뢰
├─ 박세연: 실무 협력
├─ 정하린: 대립
└─ 엘리엇 카르테인: 감시 대상 / 평가자
```

## 5. 다음 조정 필요사항

- 등급 체계는 일반, 고급, 희귀, 전설 기준으로 우선 사용한다.
- 카드 UI에서는 등급별 테두리와 아우라를 다르게 적용한다.
- 추후 캐릭터 1000개 생성 전, 이 5명을 기준 템플릿으로 삼는다.
