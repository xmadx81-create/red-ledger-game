
## 1. CARD-HUM-001-L-0001 / 서윤

```text
종족: 인간
성별/연령: 여성 청년
직업: 격투가
역할: 근접 체술 딜러
성향: GOOD
등급: L
아우라: 전설 카드 기준 골드 우선, 선 성향 기준 푸른 질서감 보조
핵심 실루엣: 정면 돌파형 인간 영웅
시각 키워드: 단단한 자세, 체술, 의지, 전설 카드 표본
주의: 실제 기관 로고 금지
```

서윤은 인간 진영의 상징적 전설 카드다. 초반 세계관의 정통파 영웅으로 사용하며, 일반 인간이 한계를 돌파하는 이미지를 담당한다.
