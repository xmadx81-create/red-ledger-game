## 목적

`헌혈의 집`의 표면 운영을 실제 서비스 현장처럼 살아 있게 만들기 위한 NPC 운영 시스템입니다.

간호사 NPC는 기본적으로 밝고 상냥한 서비스직 마인드를 가지고 있지만, 각자 성격, 외모, 말투, 업무 강점, 인간관계, 피로도, MBTI 성향이 다르게 구성됩니다.

## 기본 컨셉

```text
헌혈센터의 표면은 밝고 친절하다.
간호사 NPC는 센터의 얼굴이다.
근무 편성은 3조 3교대로 엄격하게 돌아간다.
센터 운영의 재미는 사람 사이의 시너지와 갈등에서 발생한다.
```

```text
야간 특수 센터: 비밀 이벤트 많음, 피로도 관리 중요
```

## MBTI 성향 시스템

모든 주요 캐릭터와 NPC는 MBTI 성향 태그를 가집니다.

MBTI는 절대적 성격 판정이 아니라 게임 내 상호작용 태그로 사용합니다.

```text
E/I: 외향/내향 에너지 방향
S/N: 현실 처리/직관 처리
T/F: 판단 기준
J/P: 업무 구조 선호
```

## MBTI 시너지 예시

### E + I 조합

```text
장점: 응대와 집중 업무 분업 가능
위험: 소통 방식 차이로 오해 발생 가능
```

### S + N 조합

```text
장점: 현실 처리와 아이디어 발굴 균형
위험: 한쪽은 뜬구름, 한쪽은 답답함을 느낌
```

### T + F 조합

```text
장점: 규정 판단과 감정 케어 균형
위험: 냉정함/감정적이라는 상호 비난 발생 가능
```

### J + P 조합

```text
장점: 계획성과 유연성 균형
위험: 근무표/마감/규정 준수에서 충돌 가능
```
