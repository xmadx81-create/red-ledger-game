# 대형 성장 시스템 매트릭스 001

## 기록 일자

2026-06-15

## 목적

`적혈의 장부`를 단기 엔딩형 게임이 아니라 장기 성장형 게임으로 설계하기 위한 상위 시스템 매트릭스입니다.

이 문서는 탐색, 아이템, 이벤트, 전투, 인재, 캐릭터 성장, 직업, 속성, 특기, 궁극기, 업무별 특성, 유니크 버프, 전직, 승급, 던전, 합성 시스템의 기준을 정의합니다.

## 핵심 방향

```text
헌혈센터/혈연센터는 세계관 표면이다.
실제 플레이는 캐릭터 성장, 조직 확장, 스테이지 클리어, 던전 반복, 명성 경쟁, 자산 수집을 중심으로 한다.
```

## 1. 탐색 시스템

탐색은 전투 이전의 정보 수집, 자원 발견, 이벤트 발동, 인재 조우의 기반입니다.

### 탐색 유형

```text
지역 탐색
문서 탐색
인물 탐색
수송로 탐색
암시장 탐색
던전 입구 탐색
비밀 구역 탐색
```

### 탐색 결과

```text
아이템 획득
이벤트 발생
인재 발견
전투 발생
퍼즐 발견
숨겨진 스테이지 개방
명성 변화
위험도 상승
```

## 11. 유니크 버프

캐릭터, 종족, 직업, 장비, 명성 조건에 따라 고유 버프가 붙습니다.

### 예시

```text
서윤재: 행정 안정화
김도현: 야간 의료 보정
박세연: 수송 효율 상승
정하린: 언론 위험 감지
엘리엇 카르테인: 가문 명령 강화
```

## 12. 전직 시스템

전직은 레벨, 직업 숙련도, 아이템, 이벤트 조건을 요구합니다.

### 전직 구조

```text
기본 직업 → 전문 직업 → 상위 직업 → 고유 직업
```

예시:

```text
기록관 → 봉인 기록관 → 금서 해독자 → 붉은 장부 관리자
보안요원 → 야간 경비장 → 가문 집행관 → 혈연 수호자
수송원 → 야간 운송자 → 암시장 추적자 → 그림자 물류장
```

## 13. 승급 제도

승급은 캐릭터 등급을 올리는 시스템입니다.

### 캐릭터 승급

```text
N → R → SR → SSR → UR → LR → Mythic
```

### 승급 재료

```text
동일 캐릭터 조각
직업 휘장
종족 결정
혈액 코어
가문 인장
던전 보상 재료
```

## 14. 직업별/종족별 승급 버프 차별화

동일 등급으로 승급해도 직업과 종족에 따라 버프가 다르게 붙습니다.

### 직업별 예시

```text
전투원: 공격력/치명타 증가
기록관: 정보 획득량/퍼즐 성공률 증가
수송원: 보급량/회피율 증가
의무관: 회복량/생존력 증가
보안요원: 방어력/차단률 증가
```

### 종족별 예시

```text
뱀파이어: 혈 속성 강화, 야간 보정
늑대계: 속도, 추적, 근접전 보정
엘프계: 명중, 감지, 정밀 조작 보정
드워프계: 장비 강화, 방어, 제작 보정
골렘: 체력, 방어, 상태이상 저항
위자드: 주문, 해독, 퍼즐 보정
임프: 회피, 장난, 은폐, 암시장 보정
드래곤계: 광역 피해, 위압, 고위 속성 보정
```

## 20. 합성 시스템

### 아이템 합성

```text
동일 등급 아이템 합성
재료 아이템 합성
속성 결정 합성
세트 장비 합성
금지 유물 합성
```

### 캐릭터 합성

캐릭터 합성은 조심스럽게 설계해야 합니다.

허용 방향:

```text
조각 합성
동일 캐릭터 돌파
하위 인재 계약서 합성
전직 재료 합성
```

주의 방향:

```text
주요 인물 소멸형 합성은 지양
스토리 캐릭터를 단순 재료로 소비하지 않기
```

## 21. 시스템 간 연결

```text
탐색 → 이벤트 발견
이벤트 → 인재/아이템/전투 발생
전투 → 경험치/아이템/명성 획득
던전 → 승급 재료 획득
아이템 → 캐릭터 성능 상승
캐릭터 성장 → 더 높은 난이도 개방
명성 → 새로운 지역/세력/던전 개방
```
