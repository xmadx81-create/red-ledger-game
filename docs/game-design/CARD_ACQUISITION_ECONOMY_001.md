# 일반 카드 획득 경제 001

작성일: 2026-06-16
상위 문서: `CARD_GAME_DESIGN_001.md`, `RACE_HYBRID_FUSION_SYSTEM_001.md`
데이터: `card-acquisition.json`, `fusion-recipe-tree.json`
문서 성격: 일반(N) 카드 획득 장치 + 합성 중심 경제 철학 정형화

> 철학: **N 획득은 쉽게(합성 연료 풍족), 합성은 매우 어렵게(전설로 가는 길).** 목표는 "사람들이 합성에 미치게" 만드는 것. 카드 뽑기(pull)는 없고, 두근거림은 **합성 결과**와 **사냥 잭팟**에서만 나온다.

---

## 1. N 카드는 쉽게, 합성은 어렵게

```text
N 공급 ↑↑  (출석 + 사냥 + 퀘스트로 매일 풍족)
합성 성공 ↓↓ (대부분 실패, 천장으로만 보장)
→ 유저는 N을 계속 모아 합성에 쏟아붓는다 = "합성에 미친다"
→ 노력하면 도달 가능(개인차). 가끔 운으로 어려운 확률을 뚫고 상급 합성 성공.
```

## 2. 획득 채널 (3종)

### 2.1 출석/로그인 보상 (`attendanceRewards`)
| 주기 | 지급 |
|---|---|
| 매일 | N ×1 |
| 7일 연속 | N ×5 (보너스) |
| 월 누적 | N ×20 + R ×1 보장 |
| 복귀 | N ×10 (복귀 환영, 피로도 강매 아님) |

### 2.2 사냥(스테이지/던전) 드랍 (`huntDrops`)| 드랍 | 등급 | 확률(v0) |
|---|---|---:|
| 기본 | N ×1~3 | 85% |
| 중간 | R ×1 | 8% |
| **잭팟** | **SR 이상 ×1** | **0.05%** |

- 잭팟은 "떴을 때 환호할" 극희귀 드랍. 상위 카드는 원칙상 합성으로만 만들지만, **사냥 잭팟만 직접 드랍 예외**.
- 잭팟 내부 가중치: SR 85% / EP 14.5% / L 0.5%.
- **구현 상태**: 전투 프로토타입(battle-prototype)에 연동됨 — 승리 시 위 확률로 드랍을 굴려 공유 인벤토리('hbh.inventory.v1')에 적립. 검증: 오토 15전 13승, 승리당 1회만 적립, N 30→55 증가.

### 2.3 퀘스트 (`questRewards`)
| 퀘스트 | 지급 |
|---|---|
| 일일 | N ×2 |
| 주간 | N ×10 |
| 스토리 | N ×3 (+가끔 R) |

## 3. 경제 정합성 (시뮬 검증)

`npm run sim`(tools/economy-sim.mjs)이 합성트리+획득표를 그대로 돌려 **전설 1장 도달 일수**를 플레이 강도별로 측정한다.

| 플레이 강도 | 전설 1장까지(중앙값) | 일일 N 기대 |
|---|---:|---:|
| 캐주얼(10전투/일) | 약 300~400일 | ~20 |
| 보통(25전투/일) | 약 170~190일 | ~40 |
| 하드코어(60전투/일) | 약 65~70일 | ~88 |

(RNG 40회 중앙값 기준, 실행마다 약간 변동) → "**매우 오래, 그러나 노력하면 도달**, 개인차 존재"라는 설계 의도와 일치(캐주얼 ~1년, 하드코어 ~2개월). 히든 합성 트리(`HIDDEN_FUSION_TREE_001`)는 이보다 훨씬 빠른 비공개 지름길.

전부 v0 튜닝 가능. 일일 N 공급량·합성 확률·천장·중간등급 드랍을 바꾸면 `npm run sim`으로 즉시 재측정된다.
