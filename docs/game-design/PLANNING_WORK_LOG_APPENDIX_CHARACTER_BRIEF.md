# 본격 기획 단계 작업 로그 부록 - 주요 인물 설정

## 작업 주제

주요 인물 설정 작성

## 진행 내용

1. `docs/game-design/CHARACTER_BRIEF.md`를 생성했다.
2. 기존 RPG의 익숙한 구조인 등급, 역할, 능력치, 스킬, 진영, 호감도, 배신, 개인 서사를 참고하되, 전투 중심이 아니라 운영 중심으로 변형했다.
3. 주요 인물 5명을 정리했다.
   - 서윤재 실장
   - 김도현 과장
   - 박세연 팀장
   - 정하린 기자
   - 엘리엇 카르테인
4. 각 인물의 RPG식 역할과 `적혈의 장부`식 역할을 함께 정의했다.
5. 관계 상태를 신뢰, 협력, 의심, 압박, 적대, 잠복으로 정리했다.

## 결정 사항

- 기존 RPG와 비슷한 틀을 사용하되 캐릭터의 역할을 전투력이 아니라 운영 영향력으로 해석한다.
- 파티 편성은 전투 배치가 아니라 당일 업무 배치로 변형한다.
- 호감도는 충성도, 의심도, 불안도로 변형한다.
- MVP에서는 주요 인물 5명의 역할과 사건 반응을 우선 구현한다.
