# 본격 기획 단계 작업 로그

이 문서는 `red-ledger-game` 프로젝트의 본격 기획 단계 작업 이력을 기록합니다.

---

## 2026-06-15

### 작업 주제 1

게임 전체 개요 작성

### 진행 내용

1. 본격 기획 단계 첫 문서 `docs/game-design/GAME_OVERVIEW.md`를 생성했다.
2. 프로젝트명, 한국어 가제, 한 줄 콘셉트, 핵심 요약을 정리했다.
3. 장르를 경영 시뮬레이션, 선택형 스토리, 조직관리 시뮬레이션, 다크 행정 누아르로 정의했다.
4. 표면 조직과 이면 조직을 다음과 같이 정리했다.
   - 표면 조직: 백십자재단 산하 혈연센터
   - 이면 조직: 카르테인 가문 혈액관리국
5. 1차 MVP 범위를 7일 운영 구조로 확정했다.
6. 핵심 자원 8종과 주요 인물 후보 5명을 정리했다.
7. 기본 게임 루프, 실패 조건 후보, 엔딩 방향 후보를 정리했다.

### 결정 사항

- `적혈의 장부`는 다크 행정 누아르 운영 시뮬레이션으로 기획한다.
- 실제 기관명은 사용하지 않고 가상 조직명을 사용한다.
- 1차 MVP는 7일 운영 구조를 기준으로 한다.
- 공포의 핵심은 피 자체보다 행정 시스템 뒤에 숨은 비밀 구조에서 나오도록 한다.
