# 주요 인물 설정

이 문서는 `red-ledger-game` 1차 MVP에 등장할 주요 인물과, 향후 RPG식 캐릭터 시스템으로 확장하기 위한 기본 방향을 정의합니다.

## 설계 방향

`적혈의 장부`의 캐릭터 시스템은 기존 RPG의 익숙한 구조를 사용하되, 전투 중심이 아니라 행정, 은폐, 수급, 감시, 정치, 배신을 중심으로 변형합니다.

기존 RPG에서 유저가 익숙하게 받아들이는 요소는 다음과 같습니다.

```text
등급
역할
능력치
스킬
진영
호감도
배신
동료화
개인 서사
```

이 게임에서는 위 요소를 다음처럼 비틀어 사용합니다.

```text
전투력 → 운영 영향력
직업 → 조직 내 기능
마법/공격 스킬 → 행정/은폐/수급/감시 스킬
파티 편성 → 당일 업무 배치
호감도 → 충성도/의심도/불안도
적과 아군 → 상황에 따라 변하는 정치 관계
```

## 핵심 원칙

- 캐릭터는 단순한 대화 NPC가 아니라 운영 결과에 영향을 주는 변수입니다.
- 캐릭터의 등급은 희귀도뿐 아니라 정치적 무게와 위험도를 의미합니다.
- 캐릭터는 완전한 아군이나 완전한 적으로 고정하지 않습니다.
- 상황, 선택, 자원 상태에 따라 어제의 적이 오늘의 아군이 될 수 있습니다.
- MVP에서는 수집형 캐릭터 시스템 전체를 구현하지 않고, 주요 인물의 역할성과 사건 반응을 먼저 검증합니다.

## 캐릭터 공통 능력치

MVP 및 확장 시스템에서 사용할 수 있는 캐릭터 능력치 후보입니다.

| 능력치 | 의미 |
|---|---|
| 행정 | 문서 처리, 보고, 민원, 일정 정리 능력 |
| 은폐 | 흔적 제거, 기록 조정, 정보 차단 능력 |
| 수급 | 혈액 확보, 배분, 물류 조정 능력 |
| 설득 | 인물 회유, 협상, 내부 진정 능력 |
| 감시 | 암거래 단서, 배신, 언론 움직임 포착 능력 |
| 충성 | 플레이어 또는 조직에 대한 충성도 |
| 불안 | 캐릭터가 흔들리거나 사고를 일으킬 가능성 |
| 위험도 | 강력하지만 통제하기 어려운 정도 |

## 캐릭터 등급 방향

| 등급 | 의미 | MVP 적용 |
|---|---|---|
| N | 일반 직원 또는 현장 인력 | 배경 NPC |
| R | 숙련 직원 또는 협력자 | 이벤트 보조 인물 |
| SR | 핵심 실무자 | 주요 인물 후보 |
| SSR | 강력한 세력 인물 | 가문 원로, 특수 인물 |
| EX | 세계관 핵심 인물 | 장기 확장용 |

## 주요 인물 5명

1차 MVP에서는 다음 5명을 중심으로 운영합니다.

```text
서윤재 실장
김도현 과장
박세연 팀장
정하린 기자
엘리엇 카르테인
```

## CHR-001 서윤재 실장

| 항목 | 내용 |
|---|---|
| 이름 | 서윤재 |
| 직책 | 혈연센터 실장 |
| 종족 | 인간형 또는 하이브리드 가능성 보류 |
| 초기 관계 | 플레이어 본인 또는 플레이어 대리자 |
| RPG식 역할 | 지휘관 / 관리자 / 밸런서 |
| 적혈의 장부식 역할 | 장부 관리자, 중용 판단자 |
| 등급 방향 | SR |

### 캐릭터 개요

서윤재는 혈연센터의 실무 책임자입니다. 표면상으로는 공익 의료 재단의 관리자지만, 실제로는 카르테인 가문 혈액관리국과 인간 사회 사이의 완충 장치 역할을 합니다.

플레이어가 직접 서윤재를 조작하거나, 서윤재를 플레이어의 대리 인물로 사용할 수 있습니다.

### 주요 기능

- Day 시작 보고 수령
- 자원 상태 확인
- 선택지 결정
- 중용 판단
- 가문 보고 대응
- 인간 사회와 가문 사이의 균형 유지

### 능력치 방향

| 능력치 | 방향 |
|---|---|
| 행정 | 높음 |
| 은폐 | 중간 |
| 수급 | 중간 |
| 설득 | 중간~높음 |
| 감시 | 중간 |
| 충성 | 플레이어 기준 고정 |
| 불안 | 선택에 따라 변화 |

### 고유 스킬 후보

| 스킬명 | 효과 방향 |
|---|---|
| 장부 균형 | 하루 종료 시 가장 위험한 자원 하나의 악화를 소폭 완화 |
| 결재 보류 | 특정 가문 요구를 하루 지연시키지만 가문 만족도 하락 |
| 공식 문구 | 인간 신뢰 하락을 완화하지만 언론 노출을 소폭 증가 |

### 개인 서사 방향

서윤재는 자신이 관리하는 것이 단순한 센터인지, 괴물들의 생존 장치인지 계속 의심합니다. 플레이어의 선택에 따라 냉정한 행정가, 가문에 순응하는 관리자, 인간 사회를 지키려는 내부 저항자 중 하나로 기울 수 있습니다.

## CHR-002 김도현 과장

| 항목 | 내용 |
|---|---|
| 이름 | 김도현 |
| 직책 | 야간 의사 / 혈액 처리 담당 |
| 종족 | 인간 |
| 초기 관계 | 협력자 |
| RPG식 역할 | 힐러 / 연금술사 |
| 적혈의 장부식 역할 | 야간 처리자, 의료 기록 조정자 |
| 등급 방향 | SR |

### 캐릭터 개요

김도현은 혈연센터의 야간 의료 업무를 담당합니다. 혈액 보관, 처리, 폐기 기록, 비밀 진료 관련 사건에서 핵심 역할을 합니다.

겉으로는 피로한 의사지만, 실제로는 누구보다 많은 비밀을 알고 있습니다.

### 주요 기능

- 혈액 재고 보정
- 폐기 기록 조정
- 야간 사건 처리
- 의료 관련 은폐 선택지 제공
- 폭주 직전 뱀파이어 진정 이벤트 개입

### 능력치 방향

| 능력치 | 방향 |
|---|---|
| 행정 | 중간 |
| 은폐 | 높음 |
| 수급 | 중간~높음 |
| 설득 | 낮음~중간 |
| 감시 | 중간 |
| 충성 | 중간 |
| 불안 | 높아질 수 있음 |

### 고유 스킬 후보

| 스킬명 | 효과 방향 |
|---|---|
| 야간 처리 | 혈액 재고를 회복하지만 조직 불안을 증가 |
| 폐기 기록 정정 | 보안 등급을 소모해 언론 노출 상승을 억제 |
| 응급 진정 | 수요 미충족으로 인한 폭주 위험을 1회 완화 |

### 개인 서사 방향

김도현은 자신이 의사인지, 가문의 사육사인지 혼란을 느낍니다. 계속 무리한 지시를 받으면 내부 고발자가 될 수 있고, 반대로 플레이어가 적절히 보호하면 가장 신뢰할 수 있는 실무자가 됩니다.

## CHR-003 박세연 팀장

| 항목 | 내용 |
|---|---|
| 이름 | 박세연 |
| 직책 | 물류 협력자 / 외부 운송 라인 담당 |
| 종족 | 인간 또는 하이브리드 가능성 보류 |
| 초기 관계 | 협력자이지만 완전한 아군은 아님 |
| RPG식 역할 | 로그 / 상인 / 정찰자 |
| 적혈의 장부식 역할 | 물류 브로커, 암거래 추적자 |
| 등급 방향 | SR |

### 캐릭터 개요

박세연은 혈액 운송과 외부 협력망을 관리합니다. 합법 물류와 비공식 경로를 모두 알고 있으며, 암거래망의 단서를 가장 먼저 포착할 수 있는 인물입니다.

하지만 그만큼 다른 세력과도 접점이 많아, 신뢰와 의심이 동시에 필요한 캐릭터입니다.

### 주요 기능

- 혈액 수급 보정
- 암거래 단서 발견
- 외부 거래 선택지 제공
- 물류 사고 처리
- 내부자 색출 이벤트 연결

### 능력치 방향

| 능력치 | 방향 |
|---|---|
| 행정 | 낮음~중간 |
| 은폐 | 중간 |
| 수급 | 높음 |
| 설득 | 높음 |
| 감시 | 높음 |
| 충성 | 낮음~중간 |
| 불안 | 중간 |
| 위험도 | 높음 |

### 고유 스킬 후보

| 스킬명 | 효과 방향 |
|---|---|
| 조용한 회수 | 암거래망에서 혈액 일부를 회수하지만 보안 등급 소모 |
| 뒷문 계약 | 혈액 재고를 확보하지만 암거래 단서와 언론 노출 위험 증가 |
| 운송 경로 변경 | 수급 사고를 막지만 가문 자금 소모 |

### 개인 서사 방향

박세연은 플레이어에게 유용하지만 위험한 인물입니다. 돈, 생존, 의리, 과거의 빚 사이에서 움직이며, 선택에 따라 충실한 파트너가 되거나 암거래망과 손잡는 변수가 될 수 있습니다.

## CHR-004 정하린 기자

| 항목 | 내용 |
|---|---|
| 이름 | 정하린 |
| 직책 | 탐사 기자 |
| 종족 | 인간 |
| 초기 관계 | 외부 위협 |
| RPG식 역할 | 레인저 / 추적자 / 심문관 |
| 적혈의 장부식 역할 | 언론 감시자, 폭로 위험, 잠재적 협력자 |
| 등급 방향 | SSR |

### 캐릭터 개요

정하린은 혈연센터의 야간 출입 기록, 폐기 혈액 기록, 익명 제보를 추적하는 기자입니다. 처음에는 명백한 적대 인물처럼 보이지만, 암거래망의 실체를 알게 되면 플레이어와 일시적으로 이해관계가 맞을 수 있습니다.

### 주요 기능

- 언론 노출 증가 요인
- 기록 불일치 추적
- 암거래망 외부 압박
- 플레이어의 은폐 방식에 따른 관계 변화
- 특정 루트에서 임시 협력자 전환 가능

### 능력치 방향

| 능력치 | 방향 |
|---|---|
| 행정 | 중간 |
| 은폐 | 낮음 |
| 수급 | 낮음 |
| 설득 | 높음 |
| 감시 | 매우 높음 |
| 충성 | 해당 없음 또는 낮음 |
| 불안 | 낮음 |
| 위험도 | 매우 높음 |

### 고유 스킬 후보

| 스킬명 | 효과 방향 |
|---|---|
| 집요한 추적 | 언론 노출을 증가시키고 보안 등급을 압박 |
| 익명 제보 확보 | 암거래 단서를 크게 증가 |
| 조건부 침묵 | 특정 정보를 제공하면 언론 노출을 일시적으로 억제 |

### 개인 서사 방향

정하린은 단순한 악역이 아닙니다. 그녀는 혈연센터의 비밀을 밝히려 하지만, 인간 암거래망이 더 큰 재앙이라는 사실을 알게 되면 플레이어와 불안정한 협력 관계를 맺을 수 있습니다.

## CHR-005 엘리엇 카르테인

| 항목 | 내용 |
|---|---|
| 이름 | 엘리엇 카르테인 |
| 직위 | 카르테인 가문 원로 |
| 종족 | 뱀파이어 |
| 초기 관계 | 상위 명령권자 / 압박자 |

### 캐릭터 개요

엘리엇 카르테인은 카르테인 가문의 원로로, 혈연센터를 가문의 혈액 수급 장치로 봅니다. 그는 플레이어를 보호하기도 하지만, 필요하다면 언제든 희생시킬 수 있는 존재입니다.

### 주요 기능

- 가문 수요 증가
- 가문 만족도 평가
- 강제 지시 이벤트
- 가문 정치의 압박 전달

| 능력치 | 방향 |
|---|---|
| 행정 | 중간 |
| 은폐 | 매우 높음 |
| 수급 | 높음 |
| 설득 | 매우 높음 |
| 감시 | 높음 |
| 충성 | 가문에만 충성 |
| 불안 | 낮음 |
| 위험도 | 매우 높음 |

### 고유 스킬 후보

| 스킬명 | 효과 방향 |
|---|---|
| 원로의 명령 | 특정 Day의 혈액 수요를 강제로 증가 |
| 고대 계약 | 큰 위기를 한 번 막지만 이후 더 큰 대가 발생 |

엘리엇은 단순한 보스가 아니라 오래된 질서의 대표자입니다. 그는 인간 사회를 하찮게 보지만, 동시에 너무 노골적인 폭주가 가문 전체를 위험하게 만든다는 사실도 알고 있습니다. 플레이어가 중용을 잘 지키면 인정하지만, 약점을 보이면 통제권을 빼앗으려 합니다.

## 관계 변화 설계

각 인물은 고정된 아군/적이 아니라 관계 상태를 가질 수 있습니다.

| 관계 상태 | 의미 |
|---|---|
| 신뢰 | 플레이어 선택을 지지하고 추가 선택지를 제공 |
| 협력 | 조건부로 도움 제공 |
| 의심 | 플레이어 행동을 감시 |
| 압박 | 요구를 강제하거나 자원을 소모시킴 |
| 적대 | 사건을 악화시키거나 위기를 유발 |
| 잠복 | 겉으로는 협력하지만 배신 가능성 존재 |

## RPG와의 차별점

기존 RPG와 비슷한 요소는 사용하지만, 차별점은 다음과 같습니다.

| 기존 RPG | 적혈의 장부식 변형 |
|---|---|
| 전투 파티 | 당일 업무 배치 |
| 공격력/방어력 | 행정/은폐/수급/감시 능력 |
| 힐러 | 혈액 처리와 폭주 진정 담당 |
| 로그 | 암거래 추적과 비공식 거래 담당 |
| 보스 | 상위 정치 압박자 |
| 호감도 | 충성도, 의심도, 불안도 |
| 퀘스트 | 보고서, 민원, 사건, 가문 명령 |
| 장비 | 보안 장비, 기록 장치, 출입 권한, 봉인 문서 |

## MVP 반영 범위

1차 MVP에서는 다음만 반영합니다.

- 주요 인물 5명 등장
- 각 인물의 고유 역할 표시
- 선택지에 인물 반응 연결
- 일부 선택지에 인물별 보정 효과 적용
- 관계 변화는 단순 상태값으로만 표현

등급, 성장, 스킬 레벨, 외형 수집, 개인 서사 루트는 MVP 이후 확장합니다.
