# 캐릭터 비주얼 다양성 가이드 001

## 목적

캐릭터 도감이 동일한 인상, 동일한 구도, 동일한 색감으로 반복되는 문제를 막기 위한 비주얼 다양성 규칙이다.

이 가이드는 `CARD_PRODUCTION_PIPELINE_001.md`의 원칙을 유지한다.

```text
1캐릭터 = 1이미지 = 1도감 카드
```

## 최근 문제

최근 생성 결과가 다음 방향으로 과도하게 반복되었다.

```text
젊은 미형 남성
은발/흑발 뱀프
보라 아우라
고딕 귀족 의상
상반신 클로즈업
마법 원형 후광
어두운 배경
비슷한 손 포즈
```

이 패턴은 도감 규모가 커질수록 캐릭터 구분성을 떨어뜨린다.

## 필수 다양성 축

캐릭터 생성 전 다음 7개 축을 반드시 다르게 고려한다.

| 축 | 예시 |
|---|---|
| 연령 | 소년, 청년, 성인, 장년, 노년 |
| 체형 | 마른형, 평균형, 근육형, 중후한 체형, 작은 체구, 거구 |
| 직업 실루엣 | 구조복, 방호복, 연구복, 정장, 갑옷, 공학 장비, 사제복, 야전복 |
| 색상 | 푸른 의료톤, 녹색 회복톤, 붉은 적대톤, 보라 중립톤, 금색 전설톤, 흙색 공학톤 |
| 구도 | 정면, 3/4, 측면, 전신, 무릎 위, 동작 중, 장비 사용 중 |
| 배경 | 센터 로비, 야간 보관실, 구조 현장, 연구실, 공방, 숲, 터널, 기록실 |
| 성격 인상 | 온화함, 거칠음, 노련함, 장난기, 차가움, 카리스마, 불안정함 |

## 연속 생성 제한

다음 조건은 연속 2회 이상 반복하지 않는다.

```text
뱀프 + 보라 아우라 + 고딕 귀족복
젊은 미형 남성 클로즈업
은발 남성 마법사
검은 배경 + 보라 마법진
상반신 정면 포즈
```

## 종족별 차별화 방향

### 인간

```text
현장성, 구조 장비, 생활감, 연령 다양성, 보호/민원/체술/방호 중심
```

### 뱀프

```text
고귀함만 반복 금지. 의료 협력자, 기록관, 은폐 담당, 혈향 감지자, 귀족, 야전형을 나눠 표현
```

### 엘프

```text
자연, 치유, 룬, 정령, 의식. 흰색/녹색/청색 계열 중심. 고딕 뱀프와 실루엣 분리
```

### 드워프

```text
공학 장비, 방어 설비, 공구, 보관 장치, 중후한 체형, 낮은 중심의 실루엣
```

### 하이브리드

```text
이중 속성, 불안정한 아우라, 신체 일부 변이, 복합 직업 장비
```

### 늑대인간

```text
야간 추적, 팩 가디언, 후각/발톱/기동성, 야전복, 근육형 실루엣
```

### 골렘

```text
석재/금속/수정 질감, 거대한 실루엣, 인간형 반복 금지
```

## 다음 캐릭터 제작 우선 규칙

하린처럼 기존 이미지가 계속 틀어지는 캐릭터는 다음 방식으로 재시도한다.

```text
여성 노년임을 최우선 조건으로 둔다.
젊은 여성 금지.
간호사 금지.
방호 전문가와 현장 방패 이미지를 강조한다.
마법사/뱀프/고딕 요소 금지.
푸른 아우라는 보호막 또는 투명 방패 형태로 표현한다.
```

## 생성 전 체크리스트

```text
1. 이번 캐릭터가 이전 2장과 종족/성별/연령/실루엣이 다른가?
2. 같은 색감과 배경이 반복되지 않는가?
3. 직업 장비가 캐릭터 설정과 맞는가?
4. 이미지 안에 잘못된 글자/로고가 들어갈 위험이 없는가?
5. 도감 카드에 넣었을 때 한눈에 구분되는가?
```
