# 종족·세력·정치 관계 시스템 노트

이 문서는 `red-ledger-game`의 장기 확장 세계관을 위한 종족, NPC 유형, 세력 관계, 정치적 협력/대립 구조를 정리합니다.

## 작성 배경

사용자는 NPC의 종류, 적과 아군, 스파이, 인간형, 뱀파이어, 하이브리드 뱀파이어, 늑대인간, 하이브리드 늑대, 엘프, 하이브리드 엘프, 드워프, 하이브리드 드워프, 골렘, 위자드, 임프, 광신병형 전투원, 용기병형 기계병, 드래곤 등 다양한 종족과 정치적 협력/대립 구조를 제안했습니다.

이 아이디어는 1차 MVP에 전부 넣는 것이 아니라, 장기 세계관 확장과 캐릭터 수집/세력전/스토리 분기 시스템의 기반으로 관리합니다.

## 핵심 방향

`적혈의 장부`의 세계는 단순히 뱀파이어 가문 하나만 존재하는 세계가 아닙니다.

겉으로는 인간 사회의 공익 재단과 행정 조직이 존재하지만, 그 이면에는 다양한 비인간 종족, 혼혈 종족, 마법적 존재, 고대 세력, 비밀 군사 조직, 독립 암시장 세력이 얽혀 있습니다.

플레이어는 이들 사이에서 혈액 수급망을 운영하고, 협상하고, 속이고, 감시하고, 때로는 어제의 적을 오늘의 아군으로 받아들여야 합니다.

## 설계 원칙

- 1차 MVP에서는 모든 종족을 등장시키지 않습니다.
- 종족과 세력은 장기 확장용 세계관 엔진으로 준비합니다.
- 처음에는 뱀파이어, 인간, 하이브리드, 암거래 세력 정도만 직접 등장시킵니다.
- 다른 종족은 문서, 소문, 의뢰, 외부 보고서, 봉인 기록으로 암시할 수 있습니다.
- 최종 명칭은 기존 IP와 충돌하지 않도록 모두 오리지널 명칭으로 재정의합니다.

## NPC 기본 분류

| 분류 | 설명 |
|---|---|
| 아군 | 플레이어 조직에 협력하는 인물 |
| 적대 세력 | 혈연센터, 카르테인 가문, 플레이어를 위협하는 인물 또는 조직 |
| 중립 세력 | 이익에 따라 움직이며 어느 편에도 고정되지 않는 존재 |
| 스파이 | 표면상 아군 또는 중립이지만 다른 세력의 정보를 수집하는 존재 |
| 이중 첩자 | 두 세력 이상에 동시에 정보를 흘리는 위험 인물 |
| 거래자 | 혈액, 정보, 보호, 은폐 수단을 거래하는 존재 |
| 감시자 | 특정 세력의 명령을 받고 플레이어를 관찰하는 존재 |

## 종족 기본 분류

| 종족 | 기본 특징 | 게임 내 역할 방향 |
|---|---|---|
| 인간 | 공식 사회와 행정 체계를 구성하는 다수 종족 | 신뢰, 민원, 언론, 암거래의 중심 |
| 뱀파이어 | 혈액 수요를 발생시키는 핵심 이면 종족 | 가문 정치, 공급 압박, 위협 |
| 하이브리드 뱀파이어 | 인간과 뱀파이어 사이의 경계 존재 | 충성 갈등, 폭주 위험, 비밀 인물 |
| 늑대인간 | 육체적 본능과 부족 질서가 강한 종족 | 폭력적 사건, 보호 계약, 대립 세력 |
| 하이브리드 늑대 | 인간 사회와 늑대 부족 사이의 중간자 | 잠입, 추적, 배신 가능성 |
| 엘프 | 오래된 지식, 생명 마법, 고립된 귀족성 | 고대 계약, 혈액 정화, 정치 협상 |
| 하이브리드 엘프 | 인간 사회에 섞여 사는 엘프계 혼혈 | 정보 중개, 의료/마법 보정 |
| 드워프 | 제작, 봉인 장치, 지하 네트워크에 강한 종족 | 보안 설비, 금고, 기계 장치 |
| 하이브리드 드워프 | 인간 도시와 지하 장인 사회를 잇는 존재 | 장비 제작, 밀수 경로, 기술 거래 |
| 골렘 | 인공 생명체 또는 계약 노동체 | 경비, 보관, 무감정한 집행자 |
| 위자드 | 마법, 계약, 기억 조작, 봉인술 사용자 | 고위 협상, 위험한 해결책 |
| 임프 | 하급 마계 계약자 또는 장난스러운 정보상 | 작은 거래, 누출, 속임수 |
| 광신병형 전투원 | 특정 교단이나 세력에 맹목적으로 충성하는 전사 | 습격, 선동, 희생적 행동 |
| 용기병형 기계병 | 중장갑 기계/마법 병기 계열 존재 | 고위 세력의 군사적 압박 |
| 드래곤 | 고대 권력, 지배, 부, 기억의 상징 | 최상위 정치 세력, 장기 확장 보스급 존재 |

## IP 안전 원칙

일부 명칭은 기존 유명 게임이나 작품과 강하게 연결될 수 있습니다.

따라서 최종 게임에서는 다음 원칙을 적용합니다.

| 사용자가 제안한 방향 | 최종 설계 시 처리 |
|---|---|
| 질럿 | 고유명으로 쓰지 않고 `광신병형 전투원`, `서약 전사`, `성혈 광신도` 등으로 재명명 |
| 드라군 | 고유명으로 쓰지 않고 `용기병형 기계병`, `봉인기병`, `고대 장갑병` 등으로 재명명 |
| 일반 판타지 종족 | 설정, 문화, 정치 구조를 오리지널로 재설계 |

## 세력 구조 방향

종족과 세력은 반드시 일치하지 않습니다.

예를 들어 모든 뱀파이어가 카르테인 가문 편은 아니며, 모든 인간이 피해자도 아닙니다. 인간 암거래상은 뱀파이어보다 더 위험한 변수가 될 수 있고, 하이브리드 뱀파이어는 가문과 인간 사회 사이에서 흔들릴 수 있습니다.

## 주요 세력 후보

| 세력 | 설명 | 관계 방향 |
|---|---|---|
| 카르테인 가문 | 혈액 수급망을 통제하는 뱀파이어 귀족 가문 | 플레이어의 상위 압박 세력 |
| 백십자재단 | 표면상 공익 재단 | 인간 사회와 이면 조직 사이의 가면 |
| 혈연센터 | 플레이어가 운영하는 핵심 공간 | 게임의 중심 운영 조직 |
| 인간 암거래망 | 인간들끼리 혈액을 몰래 사고파는 비공식 네트워크 | 제3의 위험 세력 |
| 야간 의료 협력자 | 비밀 진료와 기록 조작에 관여하는 인물군 | 협력 또는 배신 가능 |
| 늑대 부족 연맹 | 독자적 혈통과 영역을 가진 부족 세력 | 뱀파이어와 전통적 대립 가능 |
| 엘프계 생명회 | 생명, 혈액, 정화 의식과 관련된 고립 세력 | 협력 시 혈액 안정화 가능 |
| 드워프 장인조합 | 보안 장치와 봉인 설비를 만드는 지하 조직 | 자금과 계약 중심 협력 |
| 마법 계약자 회의 | 위자드와 계약 마법 사용자들의 느슨한 연합 | 위험하지만 강력한 해결책 제공 |
| 고대 용혈 의회 | 드래곤 또는 용혈 계통의 최상위 세력 | 장기 확장용 정치 축 |

## 관계 변화 시스템

이 세계에서는 영구적인 아군도, 영구적인 적도 없습니다.

관계는 다음 요인에 따라 변합니다.

- 혈액 공급 안정성
- 비밀 유지 여부
- 가문 자금 사용
- 특정 세력의 요구 수용 여부
- 암거래망 처리 방식
- 특정 캐릭터의 개인 서사 진행도
- 배신, 폭로, 회유, 계약 이벤트

## 관계 상태 예시

| 상태 | 의미 |
|---|---|
| 동맹 | 적극적으로 협력하는 상태 |
| 협력 | 필요할 때 거래 가능한 상태 |
| 중립 | 특별한 감정 없이 관망하는 상태 |
| 의심 | 플레이어의 행동을 감시하는 상태 |
| 적대 | 직접 방해하거나 공격하는 상태 |
| 잠복 | 겉으로는 협력하지만 배신 가능성이 있는 상태 |
| 종속 | 자금, 계약, 협박으로 통제되는 상태 |

## 게임 시스템 연결 방향

종족·세력 시스템은 다음 시스템과 연결됩니다.

- 캐릭터 등급과 능력치
- 개인 서사와 충성도
- 암거래 단서 추적
- 이벤트/선택지 분기
- 엔딩 분기
- 외교/협상 시스템
- 장기 업데이트용 신규 캐릭터와 신규 루트

## MVP 반영 범위

1차 MVP에서는 아래만 직접 반영합니다.

- 인간
- 뱀파이어
- 하이브리드 뱀파이어 암시
- 인간 암거래망
- 카르테인 가문
- 백십자재단/혈연센터
- 기자, 의료 협력자, 물류 협력자 등 인간형 NPC

나머지 종족과 세력은 확장 세계관으로 문서에 보존하고, 이후 시즌/챕터/업데이트에서 사용합니다.

## 장기 확장 방향

장기적으로는 다음 구조까지 확장할 수 있습니다.

```text
혈연센터 운영
→ 지역별 혈액 수급망 확장
→ 종족별 세력 접촉
→ 외교/계약/배신 시스템
→ 캐릭터 수집과 성장
→ 종족별 고유 스토리
→ 다중 엔딩과 세력전
```

## 결정 사항

- 종족과 세력은 장기 세계관 엔진으로 준비한다.
- MVP에서는 뱀파이어/인간/암거래망 중심으로 제한한다.
- 다양한 종족은 직접 등장보다 먼저 문서, 소문, 기록, 암시로 세계의 깊이를 만든다.
- 정치적 관계는 고정하지 않고 선택과 사건에 따라 변하는 구조로 설계한다.
- 기존 IP와 충돌 가능한 명칭은 최종 단계에서 반드시 오리지널 명칭으로 변경한다.
