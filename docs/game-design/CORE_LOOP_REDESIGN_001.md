# 핵심 루프 재정의 001

## 기록 일자

2026-06-15

## 핵심 수정

기존 Web MVP에서 사용하던 `엔딩`, `최종 보고서`, `7일 운영 종료` 중심 표현은 실제 목표와 맞지 않습니다.

이 게임은 끝나는 엔딩형 게임이 아니라, 캐릭터 성장과 명성, 스테이지 클리어, 집단 전투, 퍼즐, 아케이드, 전략 요소가 반복적으로 이어지는 지속형 게임입니다.

## 수정된 핵심 컨셉

```text
헌혈의 집처럼 보이는 혈연센터를 표면 세계관으로 사용한다.
그러나 실제 게임의 핵심은 캐릭터 성장, 조직 확장, 명성 확보, 스테이지 클리어, 다중 장르 미션 수행이다.
```

## 엔딩 제거

이 게임에는 전통적인 의미의 엔딩이 없습니다.

기존 표현:

```text
최종 보고서
엔딩
7일 운영 종료
```

수정 표현:

```text
운영 정산
스테이지 결과
랭크 판정
다음 스테이지 개방
시즌 진행도
```

## 핵심 플레이 루프

```text
1. 센터 운영
2. 자원 확보
3. 캐릭터 성장
4. 명성 상승
5. 미션/스테이지 선택
6. 전투/퍼즐/아케이드/전략 중 하나 또는 복합 진행
7. 보상 획득
8. 장비/스킬/조직 강화
9. 더 높은 난이도 스테이지 개방
10. 반복 성장
```

## 주요 성장 시스템

### 캐릭터 레벨업

각 캐릭터는 경험치, 레벨, 스킬, 장비, 역할 숙련도를 가집니다.

```text
레벨
경험치
스킬 포인트
역할 숙련도
장비 슬롯
특성
관계도
```

### 명성 시스템

명성은 게임 전체 진행의 핵심 축입니다.

```text
혈연센터 명성
가문 명성
인간 사회 신뢰
암시장 평판
언론 노출도
조직 위상
```

명성은 보상, 스테이지 개방, NPC 반응, 상점, 협력 세력에 영향을 줍니다.

### 스테이지 난이도별 클리어

스테이지는 난이도별로 반복 도전할 수 있습니다.

```text
Normal
Hard
Elite
Nightmare
Abyss
```

각 난이도는 보상, 적 구성, 퍼즐 조건, 제한 시간, 위험도를 다르게 가집니다.

## 장르 혼합 구조

전투는 핵심 요소 중 하나지만, 모든 스테이지가 전투 중심은 아닙니다.

스테이지 유형:

```text
집단 전투
잠입
퍼즐
아케이드 회피
보급 전략
협상
방어전
보스전
조사
탈출
수송
해킹/해독
```

## 유저 지배감

유저는 하나의 정답 경로를 따라가는 것이 아니라, 원하는 방식으로 게임을 지배해야 합니다.

플레이 스타일 예시:

```text
전투 중심 성장
운영 최적화 중심 성장
퍼즐/정보전 중심 성장
가문 권력 중심 성장
인간 사회 신뢰 중심 성장
암시장 장악 중심 성장
혼합 성장
```

## 반복 몰입 구조

목표는 사용자를 해치는 중독 설계가 아니라, 계속 성장하고 싶게 만드는 몰입형 루프입니다.

허용되는 설계:

```text
명확한 보상 목표
다음 스테이지 개방
캐릭터 성장 체감
새 스킬 해금
수집 욕구
전략 조합 실험
랭크 재도전
시즌 목표
```

피해야 할 설계:

```text
과도한 강제 접속
불합리한 패널티
과금 압박형 피로도
보상을 숨긴 확률 조작
유저를 괴롭히는 반복 노가다
```

## Web MVP 수정 방향

현재 Web MVP의 `Final Report` 화면은 삭제하거나 의미를 변경합니다.

수정 방향:

```text
Final Report → Stage Result
새 회차 시작 → 다음 스테이지
타이틀로 → 운영 본부로
7일 운영 결과 → 스테이지 정산
엔딩 플래그 → 진행 성향 / 명성 경향 / 해금 조건
```

## 이미지/화면 설계 영향

앞으로 필요한 이미지는 엔딩 이미지가 아니라 다음 자산입니다.

```text
스테이지 선택 화면
난이도 선택 화면
캐릭터 레벨업 화면
명성 랭크 화면
전투 준비 화면
퍼즐 미션 화면
아케이드 회피 화면
전략 지도 화면
보스전 결과 화면
보상 획득 화면
다음 스테이지 개방 화면
```
