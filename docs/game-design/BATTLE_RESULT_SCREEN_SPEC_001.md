# 전투 결과/정산 화면 설계 001

## 기록 일자

2026-06-15

## 목적

전투씬의 결과를 단순 승리/패배가 아니라 `헌혈의 집` 운영 철학에 맞게 정산하기 위한 화면 설계입니다.

전투 결과는 적 처치가 아니라 다음 기준을 중심으로 평가합니다.

```text
방문자 안전
비밀 노출 방지
공생 점수
센터 신뢰
시설 안정
피해 최소화
```

## 화면 이름

```text
전투 결과
작전 정산
공생 방어 결과
```

## 화면 흐름

```text
전투 브리핑
→ 추천 편성 확인
→ 액션 선택
→ 전투 진행
→ 전투 결과/정산
→ 보상 획득
→ 다음 이벤트/스테이지 영향 확인
```

## 결과 화면 주요 영역

### 1. 클리어 랭크

```text
S: 완벽한 공생 방어
A: 안정적 해결
B: 부분 해결
C: 위험한 해결
D: 운영 리스크 발생
```

### 2. 핵심 지표

```text
방문자 안전도
비밀 노출도
시설 안정도
시민 신뢰
공생 점수
```

### 3. 선택한 액션 기록

플레이어가 선택한 주요 액션을 보여줍니다.

```text
보호
제압
협상
은폐
증거 회수
응급 처치
보안 봉쇄
회피 기동
진정 안내
후퇴
```

### 4. 목표 달성 여부

```text
방문자 안전 확보
증거 회수
비밀 노출 억제
시설 피해 방지
방해 세력 무력화
```

### 5. 보상

```text
캐릭터 경험치
센터 명성
시민 신뢰
직원 사기
방해 세력 단서
아이템 상자
승급 재료
던전 입장권
```

### 6. 다음 영향

전투 결과는 다음 시스템에 영향을 줍니다.

```text
지역 센터 위험도
직원 피로도
시민 신뢰
방해 세력 활동도
다음 이벤트 발생 확률
스테이지 해금 조건
```


## 전투 결과 문구 예시

### S 랭크

```text
방문자 안전을 완벽히 지키며 상황을 진정시켰습니다.
센터 신뢰가 상승하고, 다음 스테이지에서 협상 선택지가 확장됩니다.
```

### A 랭크

```text
큰 피해 없이 상황을 안정적으로 해결했습니다.
일부 흔적은 남았지만 운영에는 지장이 없습니다.
```

### B 랭크

```text
목표는 달성했지만 일부 위험이 남았습니다.
다음 구간에서 추가 점검이 필요합니다.
```

### C 랭크

```text
작전은 종료되었으나 센터 운영 리스크가 증가했습니다.
직원 피로도와 시민 신뢰를 관리해야 합니다.
```

### D 랭크

```text
상황 수습은 되었지만 운영상 큰 손실이 발생했습니다.
다음 이벤트에서 방해 세력 활동이 강화될 수 있습니다.
```

## Web MVP 반영 항목
