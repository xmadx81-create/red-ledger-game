# 적혈의 장부 — 엔딩 설계

작성일: 2026-06-16

## 1. 엔딩 설계 목적

MVP의 엔딩은 7일 운영 결과를 플레이어에게 명확하게 돌려주는 평가 장치다.

엔딩은 단순한 성공/실패가 아니라, 플레이어가 어떤 균형을 선택했는지 보여준다.

## 2. MVP 엔딩 목록

| 엔딩ID | 엔딩명 | 타입 | 우선순위 |
|---|---|---|---:|
| END_BALANCED | 차가운 균형 | 일반 성공 | 1 |
| END_FAMILY | 가문의 충견 | 가문 성공 | 2 |
| END_PUBLIC | 너무 깨끗한 얼굴 | 공개 성공 | 3 |
| END_EXPOSED | 붉은 장부 공개 | 실패 | 99 |
| END_COLLAPSE | 내부 붕괴 | 실패 | 100 |

## 3. 엔딩별 조건

### END_BALANCED — 차가운 균형

조건:

```text
혈액 재고 40 이상
인간 신뢰 45 이상
언론 노출 50 미만
가문 만족 45 이상
```

결과:

표면 조직과 가문 요구를 모두 유지한 안정 엔딩.

후속 해금:

```text
2주차 운영 해금
```

### END_FAMILY — 가문의 충견

조건:

```text
가문 만족 75 이상
가문 위신 70 이상
```

금지 조건:

```text
인간 신뢰 20 미만
```

결과:

가문은 만족하지만 표면 조직의 균열이 커지는 권력형 엔딩.

후속 해금:

```text
원로 루트 해금
```

### END_PUBLIC — 너무 깨끗한 얼굴

조건:

```text
인간 신뢰 75 이상
언론 노출 25 미만
```

금지 조건:

```text
가문 만족 25 미만
```

결과:

외부 평판은 좋지만 가문 내부 불만이 남는 표면 안정 엔딩.

후속 해금:

```text
언론 대응 루트 해금
```

### END_EXPOSED — 붉은 장부 공개

조건:

```text
언론 노출 80 이상
또는 인간 신뢰 10 이하
```

결과:

외부 조사로 모든 위장 구조가 흔들리는 발각 엔딩.

후속 처리:

```text
재도전 경고
```

### END_COLLAPSE — 내부 붕괴

조건:

```text
조직 불안 85 이상
또는 혈액 재고 5 이하
```

결과:

조직 내부 실수와 이탈로 운영망이 무너지는 실패 엔딩.

후속 처리:

```text
내부관리 튜토리얼 강조
```

## 4. 엔딩 판정 우선순위

권장 판정 순서:

```text
1. 내부 붕괴 조건 확인
2. 발각 조건 확인
3. 가문 성공 조건 확인
4. 공개 성공 조건 확인
5. 균형 성공 조건 확인
6. 기본 실패 또는 미완성 엔딩
```

단, MVP에서는 대표 엔딩 5개만 우선 구현한다.

## 5. 다음 조정 필요사항

- 실패 엔딩이 너무 쉽게 발생하지 않도록 Day 1~3 수치를 조정한다.
- 성공 엔딩은 플레이어가 의도적으로 방향성을 택했을 때 도달 가능해야 한다.
- 균형 엔딩은 기본 굿 엔딩으로 설계한다.
