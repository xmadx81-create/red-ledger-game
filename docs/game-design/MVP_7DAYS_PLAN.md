# MVP 7일 운영 구조

이 문서는 `red-ledger-game` 1차 MVP의 Day 1~Day 7 운영 구조를 정의합니다.

## 목적

1차 MVP는 게임의 핵심 재미를 검증하는 최소 구조입니다.

검증 대상은 다음과 같습니다.

- 혈액 수요와 공급의 균형
- 들키지 않아야 하는 은폐 압박
- 과잉 대응과 방치 사이의 중용
- 인간 암거래망 단서 추적
- 주요 인물의 역할과 사건 반응
- 7일 결과에 따른 평가형 엔딩

## MVP 핵심 문장

```text
7일 동안 혈연센터를 운영하라.
비밀을 들키지 마라.
혈액 공급을 끊기게 하지 마라.
가문을 만족시키되, 인간 사회의 의심을 키우지 마라.
암거래망을 방치하지 말고, 그렇다고 공개적으로 터뜨리지도 마라.
```

## 전체 구조

```text
Day 1: 부임과 첫 보고
Day 2: 공급 부족의 징후
Day 3: 언론과 민원의 접근
Day 4: 내부 불안과 물류 균열
Day 5: 가문의 추가 요구
Day 6: 암거래망과 복합 위기
Day 7: 최종 보고와 평가
```

## 공통 일일 루프

각 Day는 다음 구조를 따릅니다.

```text
1. Day 시작 보고
2. 현재 자원 상태 확인
3. 오늘의 혈액 수요 확인
4. 공식 공급량과 부족분 확인
5. 사건 발생
6. 선택지 결정
7. 자원 변화 반영
8. 캐릭터 반응 확인
9. 암거래/언론/가문/내부 단서 갱신
10. Day 종료 보고
```

## 핵심 지표

MVP에서 우선 추적할 핵심 지표는 다음과 같습니다.

| 지표 | 의미 |
|---|---|
| 혈액 재고 | 현재 확보한 혈액량 |
| 혈액 수요 | 가문과 뱀파이어 세력이 요구하는 혈액량 |
| 인간 신뢰 | 혈연센터에 대한 인간 사회의 신뢰 |
| 언론 노출 | 외부에 비밀이 드러날 위험 |
| 조직 불안 | 내부 직원과 협력자의 동요 |
| 보안 등급 | 비밀 유지 체계의 안정성 |
| 가문 만족도 | 가문 핵심부의 만족도 |
| 암거래 단서 | 인간 암거래망을 파악한 정도 |

## Day 1: 부임과 첫 보고

### 목적

플레이어에게 세계관, 조직 구조, 기본 자원, 첫 선택을 소개합니다.

### 핵심 사건

새로운 실무 책임자로 부임한 플레이어는 첫 운영 보고를 받습니다. 겉으로는 평범한 혈연센터 운영 현황이지만, 별도 봉인 문서에는 카르테인 가문의 실제 혈액 수요가 기록되어 있습니다.

### 주요 선택 방향

| 선택 방향 | 결과 방향 |
|---|---|
| 정석 운영 | 인간 신뢰 안정, 가문 만족도 낮음 |
| 가문 우선 배분 | 가문 만족도 상승, 혈액 재고 감소 |
| 기록 조정 | 보안 등급 소모, 단기 위기 완화 |

### 도입할 요소

- 혈액 재고
- 혈액 수요
- 인간 신뢰
- 가문 만족도
- 서윤재 실장 또는 플레이어 역할
- 엘리엇 카르테인의 첫 압박

## Day 2: 공급 부족의 징후

### 목적

수요와 공급의 균형 압박을 본격적으로 보여줍니다.

### 핵심 사건

예상보다 많은 혈액 수요가 발생합니다. 공식 기록상 재고는 충분해 보이지만 실제 사용 가능한 혈액은 부족합니다.

### 주요 선택 방향

| 선택 방향 | 결과 방향 |
|---|---|
| 공식 캠페인 강화 | 인간 신뢰 상승, 언론 관심 소폭 증가 |
| 야간 처리 지시 | 혈액 재고 회복, 조직 불안 증가 |
| 가문 수요 일부 지연 | 재고 보존, 가문 만족도 하락 |

### 도입할 요소

- 김도현 과장
- 혈액 수요 초과
- 공급 부족 경고
- 뱀파이어 폭주 위험의 암시

## Day 3: 언론과 민원의 접근

### 목적

은폐와 공익 이미지 관리의 긴장감을 제시합니다.

### 핵심 사건

정하린 기자가 혈연센터의 폐기 기록과 야간 출입 기록 사이의 불일치를 포착합니다. 동시에 민원이 접수됩니다.

### 주요 선택 방향

| 선택 방향 | 결과 방향 |
|---|---|
| 공개 설명 | 인간 신뢰 유지, 보안 위험 증가 |
| 기자 회유 | 언론 노출 감소, 가문 자금 소모 |
| 내부 기록 정정 | 단기 은폐 성공, 장기 리스크 증가 |

### 도입할 요소

- 정하린 기자
- 언론 노출
- 보안 등급
- 기록 불일치

## Day 4: 내부 불안과 물류 균열

### 목적

내부 조직이 단순한 도구가 아니라 위험 요소임을 보여줍니다.

### 핵심 사건

물류 담당 라인에서 혈액 운송량과 도착량이 맞지 않는 문제가 발생합니다. 직원 일부는 야간 업무에 불만을 표시합니다.

### 주요 선택 방향

| 선택 방향 | 결과 방향 |
|---|---|
| 직원 보상 | 조직 불안 감소, 가문 자금 소모 |
| 물류 감시 강화 | 암거래 단서 상승, 내부 불안 증가 |
| 문제 은폐 | 단기 안정, 보안 등급 하락 |

### 도입할 요소

- 박세연 팀장
- 조직 불안
- 암거래 단서 첫 상승
- 내부 배신 가능성

## Day 5: 가문의 추가 요구

### 목적

가문이 단순한 후원자가 아니라 압박과 위험의 원천임을 보여줍니다.

### 핵심 사건

엘리엇 카르테인이 예정에 없던 추가 혈액 공급을 요구합니다. 이를 충족하지 못하면 가문 내부 일부 뱀파이어의 통제가 어려워질 수 있습니다.

### 주요 선택 방향

| 선택 방향 | 결과 방향 |
|---|---|
| 요구 수용 | 가문 만족도 상승, 혈액 재고 급감 |
| 요구 축소 협상 | 가문 만족도 소폭 하락, 재고 보존 |
| 대체 공급 탐색 | 암거래 단서 상승, 언론/보안 위험 증가 |

### 도입할 요소

- 가문 만족도
- 혈액 수요 급증
- 중용 판단
- 공급 부족 시 폭주 위험

## Day 6: 암거래망과 복합 위기

### 목적

그동안 쌓인 선택 결과가 복합 위기로 돌아오게 합니다.

### 핵심 사건

혈액 암거래망의 존재가 선명해집니다. 일부 혈액은 센터 기록에서 사라졌다가 외부 거래 단서로 다시 등장합니다. 동시에 기자, 내부 직원, 가문 요구가 한꺼번에 압박합니다.

### 주요 선택 방향

| 선택 방향 | 결과 방향 |
|---|---|
| 암거래망 조용한 회수 | 암거래 단서 감소, 보안 등급 소모 |
| 암거래망 이용 | 혈액 재고 회복, 장기 리스크 증가 |
| 내부자 색출 | 조직 불안 증가, 보안 회복 가능 |
| 기자에게 일부 정보 흘리기 | 암거래망 압박, 혈연센터 노출 위험 증가 |

### 도입할 요소

- 암거래망 실체
- 복합 자원 변화
- 캐릭터별 반응 차이
- 선택 누적 결과

## Day 7: 최종 보고와 평가

### 목적

7일 동안의 운영 결과를 평가하고 엔딩 분기를 제공합니다.

### 핵심 사건

플레이어는 카르테인 가문과 내부 조직, 인간 사회의 시선을 모두 마주한 상태에서 최종 운영 보고를 제출합니다.

### 평가 기준

| 기준 | 평가 방향 |
|---|---|
| 혈액 수요 충족률 | 공급 안정성 평가 |
| 언론 노출 | 은폐 성공 여부 |
| 인간 신뢰 | 표면 조직 유지 가능성 |
| 조직 불안 | 내부 붕괴 위험 |
| 암거래 단서 처리 | 수급망 통제력 |
| 가문 만족도 | 가문 내 생존 가능성 |

### 엔딩 후보

| 엔딩 | 조건 방향 |
|---|---|
| 안정 운영 | 수요/공급 균형, 낮은 노출, 낮은 불안 |
| 가문 우선 | 가문 만족도 높음, 인간 신뢰 낮음 |
| 인간 사회 우선 | 인간 신뢰 높음, 가문 만족도 낮음 |
| 공급 붕괴 | 수요 충족 실패 누적 |
| 은폐 실패 | 언론 노출 또는 보안 실패 |
| 암거래 잠식 | 암거래망 통제 실패 |
| 내부 붕괴 | 조직 불안 과다 |

## 1차 MVP의 최소 구현 조건

MVP로 인정하려면 다음은 반드시 포함되어야 합니다.

- Day 1~Day 7 진행 가능
- 매일 혈액 수요와 공급 비교
- 최소 8개 핵심 지표 변화
- Day별 핵심 사건 1개 이상
- 사건별 선택지 3개 이상
- 선택 결과에 따른 자원 변화
- 암거래 단서 지표
- 최종 평가 화면
