## 8. 다음 결함 후보

- 이벤트 발생 조건은 아직 문자열 중심이다.
- 엔딩 판정은 JSON 조건 기반으로 바뀌었지만, 우선순위 설계는 실제 플레이 테스트 후 재조정해야 한다.
- localStorage 저장은 브라우저 단일 기기 기준이므로, 계정 기반 저장은 추후 별도 설계가 필요하다.
- 인물 카드 초상화는 아직 임시 슬롯이므로 실제 이미지 에셋으로 교체해야 한다.
