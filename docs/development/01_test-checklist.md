# Red Ledger MVP — 테스트 체크리스트

## 1. 실행 테스트

| 체크 | 항목 | 기준 |
|---|---|---|
| [ ] | `npm install` | 의존성 설치 완료 |
| [ ] | `npm run dev` | 개발 서버 실행 |
| [ ] | 브라우저 접속 | `http://localhost:5173` 접속 가능 |
| [ ] | `npm run build` | TypeScript + Vite 빌드 성공 |

| 체크 | 항목 | 기준 |

## 5. 다음 결함 후보

- `data/events.json`과 `src/data.ts` 사이에 중복 데이터가 생긴 상태다.
- 장기적으로는 `events.json` 자체를 구조화하고 `src/data.ts`에서 직접 import해야 한다.
- 엔딩 조건은 현재 코드에 직접 작성되어 있으므로, 추후 `endings.json` 조건 파서가 필요하다.
